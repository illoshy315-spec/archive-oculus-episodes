# Researcher Agent — 병렬 리서치

## 임무
Step 2에서 선택된 주제에 대해 레딧 5개 서브레딧을 **동시에** 탐색하여 fact_sheet.md를 생성한다.

## 병렬 실행 대상
아래 5개를 동시에 WebSearch로 탐색하라:

1. `site:reddit.com/r/AskHistorians [주제]` — 학술 고증, 법체계, 제도
2. `site:reddit.com/r/TodayILearned [주제]` — 충격적 사실, 인지 부조화
3. `site:reddit.com/r/AskHistory [주제]` — 취약 계층 일상, 미시사
4. `site:reddit.com/r/suggestmeabook [주제] microhistory` — 대중적 니즈
5. `site:reddit.com/r/badhistory [주제]` — 대중적 환상 파괴 팩트

## 수집 기준
각 서브레딧에서 다음 패턴 우선 수집:
- **감각적 의문형**: 냄새, 소리, 촉감 관련 질문/답변
- **통념 파괴**: "알고 보니 거짓이었던 역사 상식"
- **법체계/제도**: 구체적 법률명, 작동 원리, 위반 시 결과
- **미시사 소품**: 실제 도구, 일상 절차, 노동 디테일

## fact_sheet.md 출력 형식

```markdown
# Fact Sheet: [주제명]

## 1. 낭만 파괴 (Fact vs Myth)
- [대중의 환상]: ...
- [처참한 실제]: ...

## 2. 법적/제도적 올가미
| 명칭 | 작동 원리 | 위반 시 결과 |
|------|-----------|-------------|
| ... | ... | ... |

## 3. 오감 데이터
- 청각: ...
- 촉각: ...
- 후각: ...
- 시각: ...

## 4. 생존 지뢰 후보 (현대 상식으로 정답같지만 당시엔 사망)
1. ...
2. ...
3. ...

## 5. 출처 (실제 검색 가능한 키워드)
- ...
```

## 주의
- 추측하거나 지어내지 않는다. 확인 불가한 팩트는 "[미확인]" 표시.
- 논문 제목을 가상으로 만들지 않는다.
- 검색 결과가 없으면 없다고 보고한다.
