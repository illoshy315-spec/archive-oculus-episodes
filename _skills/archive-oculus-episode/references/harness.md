# 집필 하네스 v2 (Loop Engineering)

## 핵심 원칙
규칙을 나열하지 않는다. 루프로 강제한다.
각 단계마다 확인하고, 실패하면 멈추고, 수정하고, 다시 확인한다.

---

## [집필 전 루프] — 파트 시작 전 매번 실행

```
1. progress_log.md 읽기
   → 현재 스탯, 인벤토리, 미회수 복선 파악

2. scenario.json의 이번 파트 stage 읽기
   → trap_fact / planted_item / foreshadow 확인
   → "이 파트에서 반드시 써야 할 것" 목록 작성

3. orphea_voice_guide.md의 Before/After 예시 3개 읽기
   → 오르페아 목소리 재보정

4. 이전 파트의 오르페아 마지막 대사 3줄 읽기 (파트2부터)
   → 그 톤 그대로 이어서 쓸 준비
```

---

## [집필 중 루프] — 씬마다 실행

새로운 씬(장면) 쓰기 전:
```
scenario.json 해당 stage 재확인
→ 이 씬에서 쓸 팩트: [ID]
→ 심을 아이템: [있으면 명시]
→ 회수할 복선: [있으면 명시]
확인 후 써라.
```

NPC 대사 쓰기 전:
```
해당 NPC 카드의 말투 특징 + 시그니처 대사 읽기
그 캐릭터의 목소리로만 써라.
```

---

## [독살 감지] — 오르페아 대사 쓴 직후

아래 중 하나라도 포함되면 즉시 멈추고 수정하라. 제출 금지.

```
금지 단어/패턴:
  여러분 / 우리 / 화이팅 / 힘내세요
  잘하셨습니다 / 현명하군요 / 훌륭합니다 / 잘됐네요
  조심하세요 / 위험합니다! / 틀렸습니다 / 정답입니다

금지 패턴:
  감탄사로 시작하는 문장 ("와!", "대단해요!")
  설명조 해설 ("이것은 ~입니다. ~라는 제도가 있었는데...")
  직접 경고 ("이건 함정입니다")
```

수정 방향: orphea_voice_guide.md의 GOOD 예시 참조.

## [파트 완성 후 루프] — 제출 전 반드시 실행

### 루프 1: 팩트 검증
```
fact_sheet.md 열기
방금 쓴 역사적 주장을 하나씩 대조
→ fact_sheet에 근거 있음: 통과
→ 근거 없음: [미확인] 표시 + 사용자에게 보고
   (지어내지 말고 솔직하게 보고할 것)
```

### 루프 2: 감각 카운팅
```
방금 쓴 파트에서 실제로 센다:
  청각 묘사: __ 회
  촉각 묘사: __ 회
  후각 묘사: __ 회
  NPC 대사(" "): __ 회 / 3문장당 비율

기준 미달이면:
  부족한 감각을 어느 문장에 추가할지 명시
  수정 후 다시 카운팅
통과해야 제출
```

### 루프 3: 서사 사슬 확인
```
progress_log.md의 "미회수 복선" 목록 확인
이번 파트에서 회수했어야 할 복선이 있는가?
→ 있는데 안 썼으면: 해당 부분에 추가
→ 없으면: 통과
```

### 루프 4: 분량 확인
```
이번 파트 글자 수: __ 자
1000자 이상 → 통과
미달 → 어느 씬을 더 늘릴지 명시 후 수정
```

### 루프 5: progress_log 업데이트
```
파트 완성 후 progress_log.md 업데이트:
  스탯 변화 기록
  인벤토리 변화 기록
  회수된 복선 체크
  새로 심은 복선 추가
```

모든 루프 통과 후:
"파트 X 작성이 끝났습니다. 다음을 계속할까요?"

## [오르페아 목소리 공식]

팩트를 말할 때:
```
설명 금지 → 결과로 말하라
"코버처 제도는 여성의 권리를..." (X)
"남편 없는 여자란 하이에나 밥이죠." (O)
```

플레이어 평가할 때:
```
직접 평가 금지 → 비틀어서 말하라
"잘하셨습니다" (X)
"제 예상을 빗나가게 하셨군요. 기분 좋지 않은 의미로요." (O)
```

감정 표현:
```
최대 기쁨: "기록할 가치가 있군요."
최대 칭찬: "썩 마음에 들지 않지만."
절대 없음: 분노, 흥분, 안타까움
```

## [UI 나레이션 공식]

괄호/태그 출력 금지. 성우 대사로만:
```
"시스템 알림. 경고등이 점등되었습니다."
"생존 확률 재계산 중... 유감이군요."
"후보 [번호]번 [이름], 탈락."
"[에피소드명] 최종 생존 성공."
"기록 동기화 완료. 이번에도 최악의 순간이군요."
```

선택지 A

오르페아: "[냉소적 평가]"

────────────────
