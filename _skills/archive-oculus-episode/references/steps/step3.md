# Step 3 — 지뢰 설계 + NPC + scenario.json 골격

## 임무
fact_sheet의 팩트들을 플레이어가 밟을 함정(지뢰)으로 설계하고
NPC를 설계하고 scenario.json 골격을 생성한다.

---

## 파트 1: 지뢰 후보 6개 설계

fact_sheet의 팩트 각각에 대해:
- 현대 상식으로는 정답처럼 보이는 선택지 설계
- 그것이 왜 당시에는 치명적이었는지 사료 근거

| 번호 | 지뢰 명칭 | 유혹적 선택 | 실제 파멸 메커니즘 | 사료 근거 | 흥미도 |
|------|-----------|-------------|-------------------|-----------|--------|
| 1 | | | | | /10 |
| 2 | | | | | /10 |
| 3 | | | | | /10 |
| 4 | | | | | /10 |

→ 사용자가 4개 선택할 때까지 대기.

---

## 파트 2: 플레이어 신분 결정 (4개 지뢰 확정 후)

**순서 원칙: 신분을 먼저 정하지 않는다.**
지뢰 4개가 확정된 후, 그 지뢰들이 모두 자연스럽게 등장할 수 있는 상황을 역으로 설계하고,
그 상황을 커버하는 신분을 선택한다.

질문: "이 4개의 지뢰가 한 에피소드 안에서 자연스럽게 발생하려면 플레이어가 어떤 위치에 있어야 하는가?"
→ 신분 후보 2~3개 제안 → 사용자 선택

## 파트 2.5: 포맷 결정 (추가)

`references/episode_formats.md` 참고하여 이 에피소드의 게임 포맷 확정.
포맷에 따라 Step 5 빌드 템플릿이 달라지므로 반드시 이 단계에서 결정.

기본형 / 법정물 / 미연시 / 육성 / 로그라이크 / 추리물 / 탈출어드벤처 / 타이쿤

## 파트 2.6: 고정 NPC 등장 여부 결정 (추가)

`references/fixed_npc_cards.md` 참고.
- 역병의사: 이번 에피소드에 등장하는가? 어떤 가면/방식으로?
- 네피아: 복선을 심을 단계인가? (오르페아 말투 변화 등)
- 글리치 NPC 후보: 에피소드 NPC 중 나중에 타락시킬 후보가 있는가?
→ 강제 아님. 등장 안 해도 무방.

## 파트 4: NPC 카드 작성

`references/npc_card_template.md` 형식으로 등장인물마다 작성.
최소 3명: 빌런1 / 빌런2(또는 최종보스) / 조력자 1명.
달콤해 보이는 캐릭터가 함정, 차가워 보이는 캐릭터가 조력자.

---

## 파트 5: scenario.json 골격 생성

```json
{
  "episode": "[에피소드명]",
  "era": "[연도]",
  "location": "[장소]",
  "player_role": "[신분/직업]",
  "stats": [
    {"id": "faith", "name": "신앙", "max": 100, "initial": 80}
  ],
  "facts": [
    {
      "id": "[팩트ID]",
      "name": "[팩트명]",
      "content": "[한 줄 설명]",
      "role": "trap | escape | background",
      "stage": [스테이지번호]
    }
  ],
  "npcs": [
    {
      "id": "[npcID]",
      "name": "[이름]",
      "motivation": "[동기]",
      "role": "villain | ally | boss",
      "appears_in": [파트번호 배열]
    }
  ],
  "stages": [
    {
      "id": 1,
      "title": "[스테이지 제목]",
      "trap_fact": "[팩트ID]",
      "planted_item": null,
      "foreshadow": null,
      "choices": [
        {"id": "A", "text": "[선택지]", "survival_rate": 5, "outcome": "bad_ending_1"},
        {"id": "B", "text": "[선택지]", "survival_rate": 65, "outcome": "stage_2"}
      ]
    }
  ]
}
```
