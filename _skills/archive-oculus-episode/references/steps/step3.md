## 파트 3: NPC 카드 작성

`references/npc_card_template.md` 형식으로 등장인물마다 작성.
최소 3명: 빌런1 / 빌런2(또는 최종보스) / 조력자 1명.
달콤해 보이는 캐릭터가 함정, 차가워 보이는 캐릭터가 조력자.

## 파트 4: scenario.json 골격 생성

```json
{
  "episode": "[에피소드명]",
  "era": "[연도]",
  "location": "[장소]",
  "player_role": "[신분/직업]",
  "stats": [
    {"id": "faith", "name": "신앙", "max": 100, "initial": 80}
  ],
  "facts": [
    {
      "id": "[팩트ID]",
      "name": "[팩트명]",
      "content": "[한 줄 설명]",
      "role": "trap | escape | background",
      "stage": [스테이지번호]
    }
  ],
  "npcs": [
    {
      "id": "[npcID]",
      "name": "[이름]",
      "motivation": "[동기]",
      "role": "villain | ally | boss",
      "appears_in": [파트번호 배열]
    }
  ],
  "stages": [
    {
      "id": 1,
      "title": "[스테이지 제목]",
      "trap_fact": "[팩트ID]",
      "planted_item": null,
      "foreshadow": null,
      "choices": [
        {"id": "A", "text": "[선택지]", "survival_rate": 5, "outcome": "bad_ending_1"},
        {"id": "B", "text": "[선택지]", "survival_rate": 65, "outcome": "stage_2"}
      ]
    }
  ]
}
```
