# Step 5 — HTML5 게임 빌드

## 임무
vn_script.md를 브라우저에서 바로 실행되는 HTML5 비주얼 노벨로 구현한다.
단일 `game.html` 파일로 출력. 외부 라이브러리 CDN만 허용.

---

## 기술 스택
- HTML5 + CSS3 + Vanilla JS
- 폰트: Google Fonts (Noto Serif KR 또는 Nanum Myeongjo)
- 이미지: 경로 변수로 처리 (레오나르도 에셋 교체 용이하게)
- 사운드: Web Audio API (선택적)

---


---

## 구현 순서

1. vn_script.md 전체 읽기
2. 씬 데이터를 JS 객체 배열로 파싱
3. HTML 뼈대 + CSS 스타일 작성
4. 씬 렌더러 함수 구현 (대사 출력 → 클릭 → 다음 대사)
5. CHOICE 분기 처리
6. STAT 변화 UI 처리
7. BAD_END / GOOD_END 처리
8. 이미지 플레이스홀더 처리 (에셋 없어도 텍스트로 동작)
9. game.html 단일 파일로 저장

---

## 이미지 플레이스홀더 규칙
에셋이 없을 때:
- 배경: 단색 그라데이션 (다크 그레이 계열)
- 캐릭터: 이름 텍스트 표시
- 나중에 레오나르도 에셋 경로만 교체하면 즉시 적용
