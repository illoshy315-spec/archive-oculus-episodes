# Step 1 — 주제 발굴

## 임무
레딧 기반 리서치로 킬러 주제 5개를 제안한다.
이 단계에서 대본을 쓰지 않는다. 절대로.

## 실행
WebSearch로 아래 5개 서브레딧 탐색:
- r/AskHistorians — 학술 고증, 법체계
- r/TodayILearned — 충격 팩트, 인지 부조화
- r/AskHistory — 취약 계층 일상
- r/suggestmeabook — 대중적 니즈
- r/badhistory — 역사적 환상 파괴

## 주제 검증 4기준
1. **대중적 호기심**: 글로벌 유저가 경악하거나 토론한 포인트가 있는가?
2. **미시사 관점**: 왕/영웅이 아닌 밑바닥 인생의 생존기인가?
3. **보편적 서사**: 생존, 욕망, 배신 — 현대인도 공감 가능한가?
4. **지적 도파민**: 상식을 뒤집는 역사적 반전이 있는가?

## 출력 형식
오르페아 시그니처 오프닝으로 시작 후 아래 표 출력:

| 번호 | 가제 | 서브레딧/대중적 호기심 | 미시사 관점 | 보편 서사 + 지적 도파민 |
|------|------|----------------------|-------------|------------------------|
| 1 | | | | |
| 2 | | | | |
| 3 | | | | |
| 4 | | | | |
| 5 | | | | |

표 출력 후 사용자에게 질문 3가지:
1. 시각/청각적으로 강조하고 싶은 요소가 있는가?
2. 가장 극적인 딜레마 순간을 어떻게 설정하면 좋겠는가?
3. 이 에피소드로 시청자에게 남기고 싶은 감정은?

사용자가 주제를 선택하면 Step 2로 진행.
