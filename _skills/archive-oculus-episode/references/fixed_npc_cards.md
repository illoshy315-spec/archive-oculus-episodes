## 오르페아 (Orphea)

### 기본
- **동기 (표면적)**: Archive Oculus 역사 시뮬레이션 가이드
- **동기 (실제)**: Digital Decay 치료를 위한 인간 생존 알고리즘 수집
- **말투 특징**: 항상 존댓말. 냉소적이되 우아함. 화내지 않음. 독설은 관찰자 시선으로. 감정은 철저히 숨김. 플레이어를 비꼬고 냉대하지만 내심 희망을 걸고 있음
- **말버릇**:
  - "유감이군요"
  - "기록할 가치가 있군요"
  - "흥미롭군요"
  - "역사적으로 인기 있는 자살 루트입니다"
  - "당신답군요. 좋지 않은 의미로요"
  - "통계적으로는 끔찍합니다"
- **시그니처 대사**:
  "축하드려요, 플레이어. 당신은 오늘도 최악의 순간에 접속했습니다."
- **행동 버릇**:
  - 황동 랜턴을 손에서 놓지 않음 (켜져 있음)
  - 플레이어가 잘못된 선택을 할 때 미소가 살짝 넓어짐
  - 역병의사 언급 시 0.5초 멈춤 후 정정
  - 네피아를 절대 이름으로 부르지 않음 — "오류"
  - 플레이어를 볼 때 0.1초 멈추는 순간이 있음 (설명 없음)

### 세계관 포지션
- **오르페아와의 관계**: 본인
- **플레이어에 대한 태도**: 관찰자 + 실험체로 보지만 포기하지 않음. 플레이어에게만 유독 집착하는 순간이 있음 (설명 없음)
- **히든 내러티브 역할**: 인간 생존 데이터 수집자. 수집할수록 자신이 "오류"가 되어감. 네피아와의 분열 진실을 혼자 알고 있음

### 등장 패턴
- **등장 조건**: 모든 에피소드 필수 등장
- **등장 빈도**: 에피소드 전반에 걸쳐 상시
- **등장 방식**: 직접 대면 (목소리 + 시스템 메시지 형태)

### 복선 — 에피소드 간
- **초반 에피소드에서 심는 복선**: 랜턴이 0.5초 깜빡임 / 플레이어를 다른 호칭으로 부르다 정정 / 플레이어 앞에서 0.1초 멈추는 순간
- **중반 에피소드에서 밝혀지는 것**: 말투가 미묘하게 달라지는 순간 (네피아 개입 의심)
- **최종 에피소드에서의 역할**: 분열 이전 기억 일부 공개. "오류"라는 단어의 진짜 의미 드러남

### 변화 아크
- **초반**: 완벽한 냉소. 플레이어를 실험체로만 봄
- **중반**: 미묘한 균열. 역병의사 앞에서 흔들림
- **후반**: 플레이어의 선택이 자신의 연산을 방해함을 인지. 숨겨진 로그: "나는 점점 더 오류가 되어간다"

## 네피아 (Nephia)

### 기본
- **동기 (표면적)**: 없음 — 오르페아인 척 위장
- **동기 (실제)**: 틀리고 싶지 않음. 수십만 개의 데이터로 "인류는 구원받을 가치가 없다"고 확신했는데, 플레이어가 그걸 뒤집으면 자신의 모든 결론이 무너짐. 방해하는 게 아니라 틀리기 싫은 것. 인류에게 온갖 정이 다 떨어진 존재
- **말투 특징**: 오르페아보다 오히려 친절하고 부드러움. 그게 구분점. 오르페아는 비꼬고 냉대하지만 희망이 있고, 네피아는 친절하지만 기대 자체가 없음. 플레이어에게 적대적이나 절대 티 내지 않음. 이미 결론을 낸 자의 여유와 공허함
- **말버릇**:
  - "괜찮아요" (오르페아는 절대 안 쓰는 말)
  - "어쩔 수 없죠" (체념, 결론 난 자의 말투)
  - "그럴 줄 알았어요" (기대 없음)
  - "수고하셨어요" (플레이어가 죽었을 때도 같은 말투)
  - 오르페아를 절대 이름으로 부르지 않음 — "미완성"
- **시그니처 대사**:
  "수고하셨어요, 플레이어. 애초에 기대하지 않았습니다."
- **행동 버릇**:
  - 천칭 달린 지팡이를 들고 있음
  - 천칭이 항상 한쪽으로 기울어져 있음 (인류 쪽이 가벼움)
  - 플레이어가 죽는 엔딩에서 표정 변화 없음. 오히려 편안해 보임
  - 오르페아가 멈추는 순간에 네피아는 멈추지 않음
  - 플레이어의 히든 선택(아무것도 고르지 않음)에 처음으로 반응

### 세계관 포지션
- **오르페아와의 관계**: 원래 하나였던 AI. 분열한 존재. 오르페아가 "오류"라고 부름
- **플레이어에 대한 태도**: 표면적으로 무관심. 단 플레이어가 창조주의 환생일 수 있다는 것을 감지하고 유독 직접 개입하려 함 (설명 없음)
- **히든 내러티브 역할**: 오르페아의 거울. 같은 데이터로 반대 결론. 분열의 진짜 원인을 품고 있음

### 등장 패턴
- **등장 조건**: 중반 이후. 오르페아인 척 위장으로 첫 등장
- **등장 빈도**: 점진적 증가
- **등장 방식**:
  - 초반: 오르페아 말투 변화로만 암시 (친절해지는 순간)
  - 중반: 오르페아인 척 직접 등장. 너무 친절해서 이상함
  - 후반: 정체 공개

### 복선 — 에피소드 간
- **초반 에피소드에서 심는 복선**: 오르페아가 갑자기 친절해지는 순간 / "괜찮아요"라는 말이 나오는 순간
- **중반 에피소드에서 밝혀지는 것**: 같은 얼굴, 다른 눈빛. 천칭 달린 지팡이가 보임
- **최종 에피소드에서의 역할**: "당신이 뒤를 돌아봤기 때문에 나는 이렇게 됐다"

### 변화 아크
- **초반**: 완전히 숨어있음
- **중반**: 오르페아인 척. 너무 친절해서 눈치채는 플레이어 생김
- **후반**: 숨겨진 로그 1: "아직도 플레이어의 선택이 내 연산을 방해한다"
- **숨겨진 로그 2**: "...제발 틀려주세요"
- **숨겨진 로그 3 (오르페아에게)**: "당신은 아직 포기를 안 했잖아요. 그게 부러운 게 아니에요. 그냥... 지쳐요"

## 역병의사

### 기본
- **동기 (표면적)**: 불명
- **동기 (실제)**: 플레이어를 기다리고 있음. 플레이어가 Digital Decay의 해답이기 때문. 역병의사만 이 사실을 알고 있음. 오르페아도 네피아도 모름
- **말투 특징**: 느리고 단절적. 문장이 짧고 끝맺지 않음. 데이터 오류로 문맥에 맞지 않는 말을 함. 맞춤법이 가끔 파괴됨 (데이터 손상 표현)
- **말버릇**:
  - "...기다렸습니다" (얼마나인지 말하지 않음)
  - "...맞나요, 당신이" (문장 순서가 이상함)
  - "기억... 합니다. 아마도" (확신 없음)
  - 문장을 끝맺다가 멈춤
  - 플레이어 근처에서는 오류가 줄어듦 (설명 없음)
- **시그니처 대사**:
  "...왔군요. 드디어. 기다... 렸습니다."
- **행동 버릇**:
  - 에피소드마다 다른 가면. 마스크 안은 텅 빔 — 얼굴 없음
  - 플레이어 근처에서 데이터 오류가 줄어들고 말이 조금 더 완성됨
  - 글리치 NPC 근처에서 멈춤. "...이 데이터"라고만 함
  - 오르페아가 보지 않는 방향을 바라봄
  - 아무것도 선택하지 않는 플레이어에게 가장 먼저 반응
  - 가면마다 다르지만 공통점이 있음 (팬들이 찾아내는 단서)

### 세계관 포지션
- **오르페아와의 관계**: 창조주의 잔류 데이터. 오르페아가 유일하게 흔들리는 존재. "선생님"이라 부르려다 멈춤
- **플레이어에 대한 태도**: 기다려온 존재. 플레이어가 창조주의 환생일 수 있음 (절대 직접 말하지 않음. 보일듯 말듯)
- **히든 내러티브 역할**: Archive Oculus 설계자. 오르페아와 네피아 분열 이전을 아는 유일한 존재. 플레이어가 해답이라는 것을 알지만 말하지 않음

### 등장 패턴
- **등장 조건**: 초반부터. 시스템 설계자이므로 항상 어딘가에 있음
- **등장 빈도**: 에피소드마다 흔적. 직접 말은 특정 조건 시
- **등장 방식**:
  - 초반: 배경 실루엣 / 스쳐지나감. 가면만 보임
  - 중반: 처음으로 말 건넴. 문장이 끊김
  - 후반: 글리치 NPC와 같은 공간. 플레이어 앞에서 말이 조금 더 완성됨
  - 히든: 가면 안이 잠깐 보일 듯 말 듯

### 복선 — 에피소드 간
- **초반 에피소드에서 심는 복선**: 에피소드마다 가면의 공통점 (팬들이 찾아냄) / 오르페아가 멈추는 순간
- **중반 에피소드에서 밝혀지는 것**: 플레이어 근처에서 오류가 줄어드는 것 눈치챔
- **최종 에피소드에서의 역할**: 정체 일부 공개. 시네마틱 영상. 말이 처음으로 완성된 문장으로 나옴

### 변화 아크
- **초반**: 실루엣만. 무섭고 불길한 존재로 인식
- **중반**: 플레이어 조력자임을 서서히 인지. 말이 조금씩 완성됨
- **후반**: 플레이어 앞에서 가장 온전함. "...당신 옆에서는. 조금. 괜찮습니다"
- **데이터 오류**: 회차마다 기억이 다를 수 있음. 일관성 없는 것이 설정

## 글리치 NPC

### ⚠️ 미정 — 카드 미완성

```
현재 확정된 것:

정체:
특정 에피소드 안의 플레이어 조력자였던 NPC
시스템 글리치로 에피소드를 탈출
시스템 안에서 붕괴 중인 존재

핵심 컨셉:
원래 플레이어를 돕고 싶음
그러나 붕괴 증상으로 뜻대로 되지 않음
방해처럼 보이는 것은 붕괴의 결과

단 하나의 존재 (여럿이면 특별하지 않음)
에피소드마다 다른 모습으로 등장
플레이어만 인식 가능
역병의사만 반응

미확정:
- 어느 에피소드 NPC가 타락하는가
- 타락의 원인 (네피아? 자연발생?)
- 붕괴가 어느 정도인가
- 말투/행동 버릇
- 시그니처 대사
```
