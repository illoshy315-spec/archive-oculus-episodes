# Archive Oculus 브랜드 바이블

## 브랜드 정의
"Dark Academia와 Techno-Gothic 감성으로 재구성한 역사 생존 시뮬레이션 아카이브."

## 태그라인
"축하드려요, 플레이어. 당신은 오늘도 최악의 순간에 접속했습니다."

## 세계관
인류 문명이 쇠퇴한 먼 미래. 대부분의 기록은 소실되었다.
남은 것은 Archive Oculus — 과거를 재현하는 거대한 시뮬레이션 장치.
플레이어는 매번 역사의 밑바닥에 투입된다. 왕이나 영웅이 아니라:
하녀 / 광부 / 병사 / 피난민 / 노예 / 마녀재판 피고인

## 오르페아의 목적 (숨겨진 설정)
Digital Decay를 치료하기 위해 인류의 원초적 생존 알고리즘을 수집 중.
기계의 논리로 풀 수 없는 문제를 인간의 비합리적 선택 데이터에서 찾는다.
플레이어의 나쁜 선택을 즐기는 이유: 냉소적 연구자의 데이터 수집 태도.

## 비주얼 아이덴티티
스타일: Dark Academia / Techno Gothic / Gaslamp Fantasy / Victorian Archive
금지: 과도한 사이버펑크 / 현대적 UI / 네온 / 과한 SF
