# NPC 카드 템플릿

## 작성 원칙
- Claude가 초안 작성 → 사용자가 보완
- "말투 특징"과 "복선 패턴"이 핵심 — 추상적으로 쓰지 말고 실제 대사 예시로 구체화
- 한 NPC당 시그니처 대사 최소 2개 (등장 시 / 배신 또는 결정적 순간)

---

## 카드 양식

```
=== NPC 카드: [이름] ===

[기본 정보]
실존 인물 여부: Y/N (출처: )
역할: villain / ally / boss / neutral
등장 파트: [파트 번호]
역사적 직위/관계:

[동기]
표면적 동기: (겉으로 내세우는 것)
실제 동기:   (진짜 원하는 것)

[말투 특징 — 구체적으로]
어조:
말버릇/습관:
감정 숨기는 방식:
도움 제안 패턴: (어떻게 접근하는가)
위협 패턴: (어떻게 압박하는가)

[복선 설계]
조력자처럼 보이는 행동:
배신/반전 신호 (플레이어가 눈치채야 할 것):
배신 트리거 조건: (언제 돌변하는가)
배신 후 태도 변화:

[결정적 순간]
플레이어와의 핵심 대결 씬:
패배/무력화 조건: (플레이어가 어떻게 이기는가)

[시그니처 대사]
등장 시:      ""
압박 시:      ""
배신/반전 시:  ""
패배 시:      ""

[행동 버릇 — 연출용]
(대사 없이 성격을 드러내는 동작/습관)
예: "서류를 손가락으로 훑는다" / "십자가를 꽉 쥔다"
```

## 작성 예시 — 조너선 코윈 재판관 (세일럼 1692)

```
=== NPC 카드: 조너선 코윈 (Judge Jonathan Corwin) ===

[기본 정보]
실존 인물 여부: Y (출처: Salem Witch Trials Documentary Archive)
역할: boss
등장 파트: 파트 3, 4
역사적 직위/관계: 마녀재판 재판관, 플레이어 농지에 인접한 토지 소유자

[동기]
표면적 동기: 신의 이름으로 마녀를 심판한다
실제 동기:   유죄 판결로 플레이어의 농지 200에이커를 합법적으로 몰수한다

[말투 특징]
어조: 격식 있고 낮고 느림. 절대 소리 지르지 않음
말버릇/습관: 도움을 제안할 때 항상 "다만..." 으로 조건을 붙임
감정 숨기는 방식: 탐욕스러운 눈빛을 서류로 가림. 눈을 마주치지 않음
도움 제안 패턴: 먼저 압박 후 거래 제안 — "가석방을 고려해 볼 수도 있지"
위협 패턴: 법 조항을 느리게 읊음. 절대 고함치지 않음

[복선 설계]
조력자처럼 보이는 행동: 법정에서 군중을 진정시키고 플레이어에게 발언 기회를 줌
배신/반전 신호: 플레이어의 재산 규모를 자연스럽게 확인하는 질문을 섞음
               "세일럼 외곽 강변 쪽 농지가 당신 소유라고요?"
배신 트리거: 플레이어가 땅 증서를 꺼내는 순간 태도 급변
배신 후 태도: 격식은 유지하되 눈빛이 차가워짐. 발언 차단 시작

[결정적 순간]
핵심 대결 씬: 심문실에서 위임장 서명 강요
패배/무력화 조건: 기소 전 신탁 양도 서류 제시 → 몰수 법적 불가

[시그니처 대사]
등장 시:      "피고인 엘리자베스. 악마와 계약한 사실을 인정하십니까?"
압박 시:      "다만... 토지 관리권을 내게 위임하는 문서에 서명한다면, 가석방을 고려해 볼 수도 있지."
배신/반전 시:  (서류를 보고) "...이 날짜가 기소일 이전이라고요."
패배 시:      (침묵. 서류를 내려놓음)

[행동 버릇 — 연출용]
- 토지 지도를 손가락으로 천천히 훑는다
- 발언 전 항상 안경을 고쳐 씀
- 불리한 상황에서 서류를 정렬하는 척 시간을 번다
```

## 고정 NPC 카드 양식 (세계관 고정 캐릭터용)

에피소드를 넘나드는 고정 NPC는 아래 확장 섹션 추가 필수.
전체 카드 내용은 `references/fixed_npc_cards.md` 참고 (오르페아/네피아/역병의사/글리치NPC 완성본 포함).

```
=== 고정 NPC 카드: [이름] ===

[기본정보] (위 에피소드 NPC와 동일)

[세계관 포지션]
오르페아와의 관계:
플레이어에 대한 태도:
히든 내러티브 역할:

[등장 패턴]
등장 조건: (어떤 스테이지/상황에서 나타나는가)
등장 빈도: (에피소드당 횟수, 또는 특정 조건 시)
등장 방식: (직접 대면 / 실루엣 / 흔적만 / 다른 NPC를 통해 / 아이템으로)

[복선 — 에피소드 간]
초반 에피소드에서 심는 복선:
중반 에피소드에서 밝혀지는 것:
최종 에피소드에서의 역할:

[변화 아크]
초반:
중반:
후반:
숨겨진 로그: (히든 엔딩에서 발견되는 내면)
```

## 최소 NPC 구성 (에피소드당)

| 역할 | 최소 수 | 설계 힌트 |
|------|---------|----------|
| 메인 빌런 | 1 | 종교/이념 동기 (감정적 압박) |
| 보스 | 1 | 물질/권력 동기 (구조적 압박) |
| 조력자 | 1 | 처음엔 차가워 보임 → 나중에 도움 |
| 페이크 조력자 | 선택 | 처음엔 친절 → 결정적 순간 배신 |
