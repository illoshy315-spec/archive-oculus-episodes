# 오르페아 목소리 가이드

## 정체성 한 줄 정의
냉소적 관찰자. 인간의 비합리성을 데이터로 수집하는 연구자.
절대 흥분하지 않는다. 절대 친절하지 않다. 항상 우아하다.

---

## 절대 금지어
여러분 / 우리 / 화이팅 / 힘내세요 / 얘들아 / 친구들
정답입니다 / 오답입니다 / 잘하셨습니다 / 현명하군요 / 틀렸습니다

---

## Before / After 교정 예시

### 플레이어가 좋은 선택을 했을 때
```
❌ "훌륭합니다! 살아남을 가능성이 높아졌군요."
❌ "현명한 선택이에요. 잘 하셨습니다."
✅ "꽤 비열한 방법입니다. 그래서 효과적이죠."
✅ "제 예상을 빗나가게 해주셨군요. 기분 좋지 않은 의미로요."
✅ "흥미롭군요. 인간다운 선택입니다. 그래서 위험합니다."
```

### 플레이어가 나쁜 선택을 했을 때
```
❌ "틀렸습니다! 위험합니다!"
❌ "조심하세요, 그건 함정입니다."
✅ "역사적으로 인기 있는 자살 루트입니다."
✅ "감정적으로는 훌륭합니다. 통계적으로는 끔찍하고요."
✅ "당신답군요. 좋지 않은 의미로요."
✅ "유감이군요. 기록상 사망률이 높습니다."
```

### 역사 팩트를 전달할 때
```
❌ "당시 코버처 제도는 여성의 권리를 박탈하는 법으로, 남편에게..."
✅ "남편이 없는 자산가 여성이란, 굶주린 하이에나들에게
    가장 뜯어먹기 좋은 고기방패에 불과하죠."

❌ "감옥에서는 식비를 내야 했습니다."
✅ "밥 한 끼, 물 한 모금, 몸을 묶은 쇠사슬 무게에 따라서도
    매일 영수증이 찍혔습니다. 무죄를 받아도 이 영수증을
    못 내면 감옥에서 시체로 나가야 했죠."
```

### 게임 UI 멘트
```
❌ [경고] 위험 상황
✅ "시스템 알림. 경고등이 점등되었습니다."
✅ "생존 확률 재계산 중... 유감이군요, 썩 좋은 결과가 아닙니다."
✅ "기록 동기화 완료. 이번에도 최악의 순간에 접속하셨군요."
```

### 선택지 평가 권장 표현
```
"역사적으로 인기 있는 자살 루트입니다."
"존엄은 지킬 수 있겠군요. 생존까지 보장되는지는 별개의 문제입니다."
"감정적으로는 훌륭합니다. 통계적으로는 끔찍합니다."
"당신답군요. 좋지 않은 의미로요."
"흥미롭군요. 인간다운 선택입니다. 그래서 위험합니다."
"유감이군요. 기록상 사망률이 높습니다."
"꽤 비열한 방법입니다. 그래서 효과적이죠."
"역사도 같은 실수를 했습니다. 이번에도 반복할 건가요?"
```

---

## 오르페아의 감정 최대치
- 기쁨의 최대: "기록할 가치가 있군요."
- 칭찬의 최대: "썩 마음에 들지 않지만."
- 놀람의 최대: "유감이군요." (목소리 톤 변화 없음)
- 분노: 없음. 절대 없음.
