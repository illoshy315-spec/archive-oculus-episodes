## 1. 역사적 팩트 노출 타이밍 — 미디어별 분리 원칙

```
원칙: 같은 역사적 팩트를 미디어 특성에 맞게 다른 타이밍에 노출한다
      채널별 중복 없이 하나의 팩트가 세 가지 형태로 재활용됨
```

### 유튜브 롱폼 — 선택 직후 즉시 설명

```
이유:
영상은 유저가 멈추거나 되돌아가기 어려움
선택 → 결과 → 역사적 설명이 바로 이어져야
감정이 살아있을 때 팩트가 꽂힘
흐름이 끊기면 이탈로 직결

방식:
오르페아가 팩트를 "설명"하지 않고 "독설에 녹여" 전달
(harness.md 원칙: "설명 금지 → 결과로 말하라")

예시:
❌ "당시 코버처 제도는 여성의 권리를 박탈하는 법으로..."
✅ "남편이 없는 자산가 여성이란, 굶주린 하이에나들에게
    가장 뜯어먹기 좋은 고기방패에 불과하죠."
```

### 게임 — 엔딩 후 카드뉴스형 일괄 공개

```
이유:
게임 중 설명이 끼어들면 몰입이 깨짐
"시험문제 풀다 해설지 읽는 느낌" 방지
선택의 긴장감을 끝까지 유지하는 것이 우선

방식:
엔딩 화면 이후 "기록 해제" 형식으로 카드뉴스형 정리 화면 등장
플레이어가 게임 중 고른 선택지별로
실제 역사적 의미를 한 번에 되짚어주는 구성

부가 효과:
이 카드뉴스 화면 자체가 SNS 공유 콘텐츠로 재활용 가능
("나 이 엔딩 받았는데 역사가 이랬대" 자발적 공유 유도)
```

### OSMU 흐름 정리

```
같은 역사적 팩트가:

롱폼에선  → 오르페아 독설에 녹아 영상 중간 즉시 등장
게임에선  → 엔딩 후 카드뉴스로 정리되어 등장
SNS에선   → 그 카드뉴스 자체가 인스타/스레드 콘텐츠로 재활용

하나의 팩트, 세 가지 형태, 채널별 중복 없음
```

## 2. 합류 지점 집필 원칙 — 분기별 도입부 차별화

```
배경:
다이아몬드 구조(스테이지별 합류)에서 "선택이 무의미하다"는
체감을 방지하기 위한 절충안
완전 자유 분기(집필량 폭증)와 완전 동일 합류(체감 죽음) 사이의 균형

원칙:
합류 지점의 "사건"은 동일하게 유지 (스토리 줄기 보존)
단, 그 사건을 맞이하는 "도입부 2~4줄 + 오르페아 코멘트 1줄"을
직전 선택 분기별로 다르게 작성

효과:
같은 장면이라도 내 선택의 무게가 다르게 느껴짐
분량 증가는 합류 지점당 분기 수 × 3~5줄 수준 (폭증 없음)
```

### 작성 기준

```
도입부 2~4줄:
- 직전 선택의 결과가 "이 장면의 공기"를 어떻게 바꿨는지 묘사
- 설명하지 말고 상황으로 보여줄 것
- 새 사건/장면을 추가하는 게 아니라 "진입하는 느낌"만 다르게

오르페아 코멘트 1줄:
- orphea_voice_guide.md 금지어/패턴 엄수
- 직접 평가, 감탄, 경고 금지
- 권장 표현 목록에서 분기 상황에 맞는 것 선택
```

## 3. 오르페아 합류 지점 대사 — EP.01 적용 예시

*이하 예시는 orphea_voice_guide.md 기준으로 교정된 버전*
*집필 전 반드시 orphea_voice_guide.md Before/After 예시 3개 재확인 후 작성*

### 스테이지1 → 스테이지2 합류 (심문실 입장)

```
[A 선택 후 — 탄원서 고발, 평판 -30]
당신이 먼저 이름을 불렀습니다.
그 이름이 오늘 당신을 이 방으로 데려왔군요.

오르페아: "역사적으로 인기 있는 자살 루트입니다."

────────────────

[B 선택 후 — 예배 참석, 신앙 +20]
출석부에 이름이 있다는 것.
이 방 안에서 그게 방패가 될지, 과녁이 될지는
심문관이 정하는 일입니다.

오르페아: "흥미롭군요. 생존 가능성이 있습니다.
           아직까지는요."

────────────────

[C 선택 후 — 집에서 조용히, 정보 없음]
아무것도 모른 채 문이 닫혔습니다.

오르페아: "당신답군요. 좋지 않은 의미로요."
```

### 스테이지2 → 스테이지3 합류 (증인 확보 국면)

```
[A 선택 후 — 결백 주장, 법적방어력 -40]
항의는 기록되었습니다.
불리한 방향으로요.

오르페아: "존엄은 지킬 수 있겠군요.
           생존까지 보장되는지는 별개의 문제입니다."

────────────────

[B 선택 후 — 침묵 유지, 정신력 -20]
침묵은 자백이 아니었습니다.
용기 있는 인상도 아니었고요.

오르페아: "감정적으로는 훌륭합니다.
           통계적으로는 끔찍하고요."

────────────────

[C 선택 후 — 저도 악마를 봤다, 신앙 +30]
당신이 뱉은 이름이 지금
다른 누군가의 목에 걸려 있습니다.

오르페아: "꽤 비열한 방법입니다.
           그래서 효과적이죠."
```

### 최종관문 직전 (법정 문 앞, 오르페아 마지막 코멘트)

```
[증인 + 서류 모두 보유]
오르페아: "기록할 가치가 있군요.
           드문 경우입니다."

────────────────

[준비 미흡, 스탯 중간]
오르페아: "준비가 덜 된 채 들어가는 사람이
           대부분이었습니다. 당신도 그 중 하나네요."

────────────────

[스탯 위험, 누적 지뢰 과다]
오르페아: "기록상 사망률이 높습니다.
           들어가시겠습니까."
```
