# ARCHIVE OCULUS UI KIT v1.0

## Design Philosophy
Archive Oculus는 일반 게임 UI가 아니다.
'역사 기록 보관소'와 '금지된 마법 장치'가 결합된 인터페이스를 지향한다.

핵심 키워드
- Dark Academia
- Techno Gothic
- Gaslamp Fantasy
- Historical Simulation
- Victorian Archive
