## 1. 포맷 카탈로그

### A. 역사 생존 시뮬레이션 (기본형)
```
정의: 선택지로 역사적 함정을 피해 살아남는 기본 포맷
대표 에피: 세일럼 마녀재판

핵심 메커닉:
- 선택지 A/B/C/D
- 스탯 (신앙, 체력, 평판 등)
- 역사적 지뢰 4개
- 엔딩 구조: 퍼펙트 생존 / 일반 생존 / 사망 + 히든 1개 (3+1 구조)
  ※ 상세 조건은 archive_oculus_game_pipeline_v1.md Step 3.5 참고

오르페아 역할: 냉소적 가이드
UI 요구사항: 아스트롤라베 스탯 / 양피지 대화창 / 랜턴 생존률

난이도: ★★★☆☆
제작 난이도: ★★☆☆☆ (기본 템플릿)
유튜브 활용: "당신이 저 시대에 태어났다면" 썸네일
```

### B. 법정물
```
정의: 증거 수집 → 논리 반박 → 무죄 입증
대표 에피: 세일럼 마녀재판 법정 / 중세 이단 심판

핵심 메커닉:
- 증거 아이템 수집
- 증언 반박 시스템
- 논리 게이지 (설득력 스탯)
- 시간 제한 (재판 날짜)

오르페아 역할: 냉소적 변호 어시스턴트
"증거를 제시하세요. 논리적으로."

UI 요구사항:
- 증거 목록 패널
- 증언 텍스트 + 반박 선택지
- 재판관 신뢰도 게이지

난이도: ★★★★☆
제작 난이도: ★★★☆☆
유튜브 활용: "진짜 역사 재판 재현" / "당신이 변호사였다면"
```

### C. 미연시 (연애 시뮬레이션)
```
정의: NPC 호감도 관리 → 누구를 믿느냐가 생존 결정
대표 에피: 베르사유 궁정 / 오스만 제국 하렘

핵심 메커닉:
- 호감도 시스템 (다수 NPC)
- 달콤한 캐릭터 = 함정
- 차가운 캐릭터 = 진짜 조력자
- 관계도 분기

오르페아 역할: 의미심장한 힌트
"감정적 선택이군요. 통계적으로 위험합니다."

UI 요구사항:
- 캐릭터 스프라이트 (표정 변화)
- 호감도 하트/게이지
- 관계도 패널

난이도: ★★★☆☆
제작 난이도: ★★★★☆ (스프라이트 필요)
유튜브 활용: "숨겨진 공략 캐릭터" / "진짜 조력자 찾기"
```

```
정의: 시간/일정 관리로 스탯을 올려 생존

UI 요구사항:
- 캘린더/날짜 표시
- 다중 스탯 바
- 일정 선택 메뉴

난이도: ★★★☆☆
제작 난이도: ★★★☆☆
유튜브 활용: "최고 엔딩 육성법" / "30일 생존 챌린지"
```

### E. 로그라이크
```

- 회차 기록

난이도: ★★★★★

유튜브 활용: "몇 번 죽어야 클리어?" / "최단 클리어 도전"
```

## 3. 에피소드 카테고리 구조

```
시대/지역/국가로 묶지 않는다
"생존 메커니즘"으로 묶는다

왜?
트렌드만 따라가면 브랜드가 안 생기고
세계관만 고집하면 조회수가 안 나온다
오르페아라는 허브가 있으니 둘 다 가져갈 수 있다

트렌드 = 에피소드 선정 기준
세계관 = 브랜드 기준
```

**카테고리**

```
A. 질병
흑사병 / 스페인 독감 / 라듐 걸스 / 기면성 뇌염 / 조선 역병

B. 노동
런던 하녀 / 탄광 소년 / 방직공장 소녀 / 고래잡이 선원

C. 재난
폼페이 / 체르노빌 / 삼풍백화점 / 런던 대화재

D. 여성 생존
빅토리아 시대 여성 / 마녀재판 피고인 / 조선 과부 / 로마 노예

E. 범죄와 법
이단 심판 / 잭 더 리퍼 / 중세 재판 / 독살 미스터리

F. 전쟁 속 민간인
30년 전쟁 / 노르망디 / 임진왜란 / 베트남 민간인
```

**초반 20편 전략**

```
카테고리를 섞어서 실험
→ 어떤 생존 메커니즘이 가장 강한 반응인지 데이터 수집
→ 반응 좋은 카테고리를 시즌/게임북으로 패키징

지금 단계에서 하나의 시대를 고정하지 않는다
다양성이 나중에 게임북/출판/비주얼노벨의 여러 루트가 된다
```

## 4. 에피소드 로드맵 (초반 예시)

```
EP.01 — 세일럼 마녀재판 (카테고리 D/E)
포맷: 기본 생존
역병의사: 첫 등장 (실루엣만)
목적: 엔진 테스트 + 세계관 도입

EP.02 — 흑사병 런던 (카테고리 A)
포맷: 로그라이크
역병의사: 두 번째 등장 (말 없이)

EP.03 — 런던 하녀 (카테고리 B)
포맷: 육성 시뮬레이션
역병의사: 처음으로 말 건넴

EP.04 — 베르사유 궁정 (카테고리 D)
포맷: 미연시
네피아: 복선 시작 (오르페아 말투 변화)

EP.05 — 이단 심판 (카테고리 E)
포맷: 법정물
네피아: 오르페아인 척 첫 등장

이후 에피소드:
카테고리 반응 데이터 보고 결정
포맷도 반응 보고 결정
```

**세계관 캐릭터 등장은 에피소드 순서와 무관하게**

```
역병의사: 초반부터 꾸준히 흔적
네피아:   중반 이후 (에피소드 4~6 사이)
글리치 NPC: 후반 (팬덤 형성 후)
글리치 스페셜: 시즌 마무리
```
