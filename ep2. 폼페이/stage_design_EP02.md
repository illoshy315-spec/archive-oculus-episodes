## 0. 분기 트리 + 엔딩 조건

```
오프닝 → S1(A/B/C) → S2(A사망/B/C) → [긴 밤: 침착-15 자동] → 진입:갈증 → S3(A/B사망/C)
      → 진입:금팔찌 → 최종관문(A사망/B/C사망) → 엔딩 정산
```

| 엔딩 | 트리거 (위에서부터 우선 판정) |
|------|------|
| **히든 "비문에 남는 이름"** | 북길 생존 + livia_saved + castor_debt + !looted_gold + 갈증≠메룸 |
| **퍼펙트 "불탄 장부"** | 북길 생존 + wealth ≥ 40 + bond ≥ 50 + !looted_gold |
| **일반 "난민"** | 북길 생존 (위 둘 미달 or looted_gold) |
| **사망 (7경로)** | S2-A 매몰 / S3-B 소강귀환 / S3-A drunk 방향상실 / 관문-A 해변 / 관문-C 남길 / 관문-B 체력 낙오 / (침착 0 공황 강제 루트) |

### 엔딩 4필드

**히든 — 비문에 남는 이름** (survivor_records 팩트)
- trigger: 위 표
- scene_beat: 수십 년 뒤 누케리아. 석공이 비문을 새긴다 — 플레이어와 리비아 가족의 이름이 나란히. 카메라가 돌 표면의 끌자국에서 끝남 (Tuck 연구의 "비문으로 생존자를 추적한다" 방법론 오마주)
- npc_resolution: 니게르 가족과 의형제 관계, 카스토르는 노새와 함께 장수 (한 줄씩)
- orphea_tone: 냉소 최소치. "돌에 이름이 남는 확률까지 계산하진 않았습니다. 기록 담당자로서, 이건 보관하겠습니다."

**퍼펙트 — 불탄 장부**
- trigger: 위 표
- scene_beat: 누케리아 등록소. 서기: "신분은?" 침묵 한 박자. "자유민." 아무도 반박할 서류가 없다 — 당신을 노예로 만들던 것은 장부였고, 장부는 재 밑에 있다. ※역사 단언 금지 — "그렇게 등록되었다"는 개연성 창작. 에필로그 팩트는 티투스 재건 위원·생존자 재정착(검증됨)으로 한정
- npc_resolution: 루키우스 일가 해변에서 전멸(오프스크린, 담담히), 카르푸스는 금고와 함께 (S2-A 미선택 시 언급만)
- orphea_tone: 의미심장한 축하. "기록이 없다는 건, 때로 최고의 기록입니다."

**일반 — 난민**
- trigger: 위 표
- scene_beat: 누케리아 길바닥 배급 줄. 이름을 물어도 증명할 것이 없다. looted_gold 분기: 배급 줄에서 니게르(또는 카스토르)와 눈이 마주침 — 그가 시선을 돌린다. 고발도 용서도 아닌 침묵 (지목 구조의 EP2 변형 — 지목자 없이 시선만)
- npc_resolution: 목격자 NPC와 관계 파탄 (대사 없음, production_spec 4.7 침묵 처리)
- orphea_tone: 독설 거둠, 건조한 1줄. "살았습니다. 오늘은 그걸로 충분하다고 해두죠."

**사망 공통** — 화면이 재로 덮이며 캐스트로 굳음 → 웹 레코드 카드 인라인 (경로별: 매몰=R01 / 소강=R05 / 해변=R02 / 남길=R07 / 메룸=R12 / 낙오=R03 / 공황=R05)
- 경로별 scene_beat은 vn_script에서 분리 집필. orphea_tone: 표준 사망 코멘트 + 해당 레코드 훅 인용

- badplay: 관문-A 해변 ("함대는 진짜입니다" — 인지부조화 최대)

## 3. 스탯 도달성 시뮬레이션 (경로별 min/max)

**튜닝 변경 2건 (시뮬레이션이 잡음):**
1. **"긴 밤" 자동 이벤트 신설** (S2 종료 시 침착 −15) — 없으면 S3-A 침착 게이트(≥50)가 맨정신 플레이어에게 도달 불가능한 죽은 게이트 (최저 경로 55). EP01 "정신력 30" 사고의 역방향 재발을 잠금 전에 차단

## 4. 아이템 명세 (1개 — 인라인)

### `bedding` — 이불과 베개
실물: 양털 채운 이불 한 채 + 베개. 재투성이.
표시: 인벤토리 아이콘 — 끈으로 묶은 침구 뭉치
설명: "머리를 지키는 물건. 오늘 밤에 한해서, 갑옷이다."
획득(S2-C): 지붕 창을 뜯다 침대를 본다. 플리니우스의 부하들이 그랬듯, 이불을 머리에 두르고 끈으로 묶는다. 재가 이불을 두드리는 소리가 빗소리처럼 바뀐다.
사용(S3·관문-B): 자동 적용 — 이동 씬 hp 비용 절감 서술 + 북길 게이트 완화. 니게르 합류 시 그가 진 이불 보따리가 동일 효과 (유대의 물화).
미보유: 게이트 원래 값(30) 적용. 선택지 회색 처리 없음 (패시브 아이템).

## 5. 에피소드 서사 개요 (요약 — vn_script 집필용 비트)

- **오프닝**: 접속 시네마틱. 경고 변주 — **"이 시뮬레이션에서 '기다림'은 사망 원인입니다."** 자산 공개: "페쿨리움, 자유까지 40데나리우스 부족. 약점: 당신 자신이 자산 목록에 있다는 것." 잔진, 개 짖는 소리, 빵 굽는 냄새. 오르페아 스포일러: "주민 대부분은 살아남습니다."
- **S1**: [청각] 광장의 웅성임/첫 굉음 [시각] 우산 소나무 구름. 카르푸스 등장(명령 전달), 루키우스 퇴장 씬(마차에 궤짝 — 사람 자리 없음). 지뢰: ordered_to_stay.
- **S2**: [촉각] 지붕의 진동 [청각] 경석 우박 소리, 문 두드림. 카르푸스 계단 대치(침착 게이트). 긴 밤(침착-15). 지뢰: shelter_trap.
- **S3**: [청각] 침묵 → 멀리 무너지는 소리 [시각] 창밖 잿빛 벌판. 갈증 이벤트. 지뢰: lull_return + merum.
- **최종관문**: [시각] 산 쪽 붉은 섬광 [촉각] 뜨거운 바람. 금팔찌. 세 갈래 길 — 해변(루키우스 재회)/북길(카스토르·노새 자리 정산)/남길. 전 스탯·아이템 회수 지점.
- **엔딩**: 위 4필드 참조.
