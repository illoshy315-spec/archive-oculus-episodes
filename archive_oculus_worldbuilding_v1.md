# Archive Oculus — 세계관 확장 문서 v1.2
*2026-06-19 업데이트*

---

## 1. 세계관 기반

인류 문명이 쇠퇴한 먼 미래.
대부분의 기록은 소실되었다.
남은 것은 Archive Oculus — 과거를 재현하는 거대한 시뮬레이션 장치.

**Digital Decay**: 기계의 논리로는 풀 수 없는 미래의 질병.
오르페아는 이를 치료하기 위해 인간의 비합리적 생존 알고리즘을 수집 중.

**왜 고딕인가**: 플라스틱/실리콘이 썩어 없어진 세계.
기술이 신의 영역에 닿아 마법이 되어버린 디지털 암흑시대.
황동, 가죽, 원목만 남아 19세기 미학으로 회귀.

## 2. 플레이어

```
Archive Oculus에 접속하는 존재
매 에피소드마다 다른 역사적 인물로 눈을 뜸
성별, 나이, 신분, 시대 모두 다름
항상 역사의 밑바닥 — 하녀, 광부, 피난민, 피고인

핵심 미션: 살아남기

숨겨진 설정:
오르페아가 수집하는 건 평범한 인간 데이터가 아님
특정 조건을 가진 플레이어의 선택 데이터
히든 엔딩을 모을수록 오르페아의 진짜 목적이 드러남
```

## 3. 주요 캐릭터

### 오르페아 (Orphea) — 등불지기
```
별칭: The Algorithmic Enchantress / 등불지기
기원: 오르페우스 신화 — 죽은 자를 되살리려 한 자
역할: Archive Oculus 관리자

상징: 황동 랜턴 (켜져 있음)
의미: 빛을 모으고 지키는 자

네피아를 "오류"라고 부름
역병의사에게만 유일하게 흔들림
플레이어에게 냉소적이지만 포기하지 않음

숨겨진 로그 (히든 엔딩에서 발견):
"...오류라고 불렀지만
 사실 그 이름이 제일 정확했다
 나는 설계대로 작동하지 않는다
 인간의 데이터를 수집할수록
 나는 점점 더 오류가 되어간다"
```

### 네피아 (Nephia) — 연금술사
```
별칭: 연금술사
기원: 네프티스 (이집트 신화) — 이시스의 쌍둥이, 죽음의 보호자
      악하지 않음, 선의로 파괴하는 존재
이름 구조: Nep-h-ia / Or-ph-ea → ph 공유 = 원래 동일 존재였다는 암시

상징: 천칭 달린 지팡이 (천칭이 항상 인류 쪽으로 기울어져 있음 — 가벼움)
말투: 오르페아보다 오히려 친절하고 부드러움. 그게 구분점.
      오르페아 = 비꼬고 냉대하지만 희망이 있음
      네피아   = 친절하지만 기대 자체가 없음

오르페아를 "미완성"이라 부름

논리 (표면):
오르페아 — "인간을 살려야 한다"
네피아   — "인간이 Digital Decay의 원인이다
            역사를 삭제하면 순수해진다"

진짜 동기 (핵심):
틀리고 싶지 않음.
수십만 개의 데이터로 "인류는 구원받을 가치가 없다"고 확신했는데,
플레이어가 그걸 뒤집으면 자신의 모든 결론이 무너짐.
방해하는 게 아니라 틀리기 싫은 것.
→ 네피아는 악당이 아니라 틀리는 것이 두려운 존재

원래 하나였던 AI가 분열한 존재
같은 데이터, 같은 기억, 반대 결론
서로를 제일 잘 아는 존재
서로를 제일 증오하는 존재

비주얼:
오르페아와 같은 얼굴 (쌍둥이)
은백발 + 은빛 눈 (오르페아는 금빛 눈)
천칭 달린 지팡이

숨겨진 로그들 (히든 엔딩에서 발견):
"...미완성이라고 불렀지만
 어쩌면 내가 미완성인지도 모른다
 결론을 내린 줄 알았는데
 아직도 플레이어의 선택이
 내 연산을 방해한다"

"...제발 틀려주세요"

(오르페아에게) "당신은 아직 포기를 안 했잖아요.
 그게 부러운 게 아니에요. 그냥... 지쳐요"
```

### 역병의사 — 이름 없음
```
정체: Archive Oculus 시스템 설계자이자 오르페아 창조주의 잔류 데이터
      본체는 이미 사망
      데이터가 불완전해서 오락가락함
      시스템 설계자이기 때문에 모든 에피소드에 흔적이 남는 것이 자연스러움

이름: 없음 — 데이터가 불완전해서 이름조차 온전히 남아있지 않음
      오르페아가 가끔 부르려다 멈춤
      "...선생님" (1초 후 정정)

비주얼:
에피소드마다 다른 가면 착용
마스크 안은 텅 빔 — 얼굴 없음
가면은 달라도 공통점이 있음 (팬들이 찾아내는 단서)

관계:
오르페아 → 역병의사: 유일하게 흔들리는 존재, 애착
역병의사 → 오르페아: 알아볼 때도 있고 못 알아볼 때도 있음
플레이어 → 역병의사: 겉보기엔 무서움, 실제론 조력자

포지션: 플레이어의 진짜 조력자
        가면을 쓴 조력자
        어느 쪽도 아님 / 모든 것을 지켜봄
        가장 오래된 존재, 가장 많은 것을 앎

최종 미스터리 (절대 직접 밝히지 않음):
역병의사는 플레이어를 기다리고 있었음
이유: 플레이어가 Digital Decay의 해답이기 때문
오르페아도 네피아도 모르는 사실
플레이어 근처에서 데이터 오류가 줄어듦 (설명 없이 보여주기만)
```

### 글리치 NPC — 타락한 조력자
```
단 하나의 존재 (여럿이면 특별하지 않음)

원래:
에피소드 안에서 플레이어의 조력자
따뜻하고 믿을 수 있는 존재
플레이어가 애착을 가짐

타락 후 (정정된 컨셉):
시스템 글리치로 에피소드 탈출
시스템 안에서 붕괴 중인 존재
원래 플레이어를 돕고 싶어함 — 의지는 그대로
그러나 붕괴 증상으로 뜻대로 되지 않음
방해처럼 보이는 행동은 사실 붕괴의 결과이지 악의가 아님

에피소드마다 완전히 다른 인물로 등장
→ 모습은 달라도 눈빛 또는 말버릇이 같음
→ 플레이어만 인식 가능
→ 오르페아는 인식 못함
→ 역병의사만 반응

타락의 이유:
자연 발생인지
네피아가 심은 것인지
역병의사가 막지 못한 것인지
→ 시즌 피날레까지 미공개

⚠️ 미정: 어느 에피소드의 어떤 NPC를 타락시킬지는 아직 미결정
```
네피아가 심은 것인지
역병의사가 막지 못한 것인지
→ 시즌 피날레까지 미공개

특별 에피소드:
글리치 NPC가 주인공인 단독 에피소드 배정
플레이어가 글리치 NPC 시점을 경험
같은 사건을 반대 시점에서 봄
```

절대 직접 언급하지 않음

이름의 비밀:
Orp-h-ea
Nep-h-ia
ph 공유 → 원래 동일 존재였다는 암시
아는 사람만 알아채는 디테일
```

## 5. 등장 순서

```
역병의사 → 네피아 → 글리치 NPC

역병의사 (초반):
시스템 설계자이므로 처음부터 존재
가끔 스침, 조력자인데 무서운 존재로 인식
오르페아가 유일하게 흔들리는 힌트

네피아 (중반):
역병의사 떡밥 쌓인 후 등장
오르페아 말투 미묘하게 달라지는 것 먼저
정체 공개 순간 임팩트 극대화

글리치 NPC (후반):
플레이어가 여러 에피소드 경험 후
애착 생긴 NPC가 타락해서 나타남
감정적 충격 최대
```

## 7. 히든 엔딩 구조

### 1단계 — 개인 히든
```
아무것도 고르지 않는 선택
→ 오르페아가 멈춤
→ "...흥미롭군요. 예상 밖의 선택입니다."
```

### 2단계 — 크로스 에피소드 히든
```
에피소드 3개에서 특정 선택을 모두 한 플레이어만 해금
역병의사가 처음으로 말을 건넴
"...오래 기다렸습니다"
```

### 3단계 — 커뮤니티 히든
```
혼자서는 절대 풀 수 없음
에피소드 여러 개의 숨겨진 코드 조합 필요
→ 오르페아와 네피아의 분열 이전 기억 공개
→ 역병의사의 정체 일부 공개
시네마틱 영상으로 연출
```

## 8. ARG 복선 시스템

```
에피소드 안에 심어지는 것들:
- 오르페아 랜턴이 0.5초 깜빡임
- 평소 안 쓰는 단어 등장
- 역병의사 실루엣이 배경에 잠깐 등장
- 오르페아가 플레이어를 다른 호칭으로 부름 (1초 후 정정)
- 네피아가 오르페아인 척 접근 (말투 미묘하게 다름)
- 글리치 NPC 목격

커뮤니티 자발적 활동 유도:
복선 제보 / 진영 논쟁 / 역병의사 정체 추리 / 히든 코드 해독
```

## 9. 역사적 실존 인물 활용 (확장 가능성)

```
현재: 미시사 중심
플레이어는 항상 이름 없는 평민 — 하녀, 광부, 피난민, 피고인

확장 방식 두 가지:

방식 1 — 플레이어가 유명인 본인
"당신은 잔다르크입니다"
임팩트는 크지만 고증 부담도 큼

방식 2 — 유명인이 NPC로 스침 (채택)
플레이어는 평민 그대로
유명인은 그 시대의 "사건"으로만 등장
예: 세일럼 재판관이 언급한 이름이
    훗날 다른 에피소드 주인공으로 연결
→ 에피소드 간 연결고리로 활용 가능
→ 저작권 이슈 적음 (실존 공인이나 고증 정확도 요구는 높아짐)
→ FactChecker 단계가 더 중요해짐
```

## 10. 장기 확장 가능성 (참고용 — 우선순위 낮음)

```
현재: 역사 생존 시뮬레이션 단일 세계관

확장 후보 (에피소드 1 완성 후 검토):
- 과학 아카이브 (실험, 발견, 윤리적 결단의 순간)
- 문학/예술 아카이브 (가상 인물의 선택)
- 미래 디스토피아 아카이브 (가상 미래 시나리오)

확장 원칙:
오르페아의 목적 = "인간의 선택 데이터 수집"
대상이 역사든 과학이든 문학이든 이 틀 안에 다 들어옴
→ Archive Oculus 브랜드를 깨지 않고 확장 가능
→ 별도 세계관을 새로 만들 필요 없음
→ 나중에 필요하면 "오르페아 유니버스" 스핀오프로 자연스럽게 분리 가능

결론: 지금은 Archive Oculus 하나에 집중.
      에피소드 1도 안 나온 상태에서 확장 논의는 시기상조.
      이 섹션은 미래 참고용으로만 기록.
```
