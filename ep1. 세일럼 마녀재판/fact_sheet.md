# Fact Sheet: 세일럼 마녀재판 1692
*Step 2 산출물 — Archive Oculus EP.01*

---

## 1. 낭만 파괴 (Fact vs Myth)

| 대중의 환상 | 처참한 실제 |
|------------|------------|
| 마녀들은 화형당했다 | 전원 **교수형**. 화형은 유럽. 미국 세일럼에서 화형은 없었음 ✅ |
| 무고한 사람들은 결백을 주장해 풀려났다 | 결백을 주장한 19명은 **전원 교수형**. 자백한 54명은 전원 살아남음 ✅ |
| 마녀재판은 세일럼에서만 일어났다 | 주변 Andover, Topsfield, Ipswich, 심지어 **메인주**까지 확산. 약 200명 기소 ✅ |
| 가난하고 소외된 자들만 고발당했다 | 부유한 지주, 교회 집사, 심지어 **총독 부인 Lady Mary Phips**도 고발됨 ✅ |
| 여성들만 희생됐다 | 남성도 20~25% 포함. Giles Corey는 돌에 눌려 압사 ✅ |

## 2. 법적/제도적 올가미

| 명칭 | 작동 원리 | 위반 시 결과 |
|------|-----------|-------------|
| **스펙트럴 에비던스** (Spectral Evidence) | 고발자가 "피고인의 유령이 나를 괴롭혔다"고 증언하면 증거로 채택. 반증 불가능 | 피고인이 아무리 알리바이를 대도 무의미 ✅ |
| **자백 = 생존** | 유죄 인정 시 처형 대기. 단, 공범 이름을 대면 처형 유예 | 자백한 54명 전원 살아남음. 결백 주장한 19명 전원 처형 ✅ |
| **플리 거부 시 프레싱** | 재판 자체를 거부하면 고문(Peine forte et dure) — 돌로 눌러 자백 강요 | Giles Corey: 3일간 돌에 눌려 압사. "More weight"라는 말만 함 ✅ |
| **Witchcraft Act of 1604** | 마법 사용을 중범죄로 규정. 두 번째 유죄 시 사형 | 피고인 선택지: 자백(공범 제보) / 재판(거의 사형) / 재판 거부(압사) |
| **재산 몰수** | 유죄 판결 시 재산이 주에 귀속 가능 | Giles Corey는 재판 전 미리 아들에게 재산 양도해 몰수 막음 ✅ |
| **임신부 예외** | 임신 중이면 출산 전까지 처형 유예 | Elizabeth Proctor — 임신으로 처형 연기됨. 단, 결국 판결은 유효 ✅ |

## 3. 오감 데이터

- **청각**: 법정 안 군중 웅성거림. 고발자 소녀들의 발작 비명. 교수대 발판 떨어지는 소리
- **촉각**: 감옥 쇠고랑 — 손목, 발목에 철제 족쇄. 벽에 쇠사슬로 고정. 흙바닥. 물기가 스며드는 벽
- **후각**: 지하 감옥 — 씻지 못한 인체 냄새 + 배설물 + 지하 흙 + 썩은 짚
- **시각**: 영구 어둠. 창문에 쇠창살. 촛불 하나 없는 공간. 오전에만 희미한 빛
- **온도**: 1692년 겨울~봄 — 뉴잉글랜드 지하 감옥은 영하 수준. 담요 없음

## 4. 생존 지뢰 후보 (현대 상식으로 정답같지만 당시엔 사망)

1. **"나는 결백합니다" — 결백 주장** ✅
   - 현대 논리: 무고하면 당연히 결백을 주장해야 한다
   - 1692 실제: 결백 주장 = 재판 → 사형. 자백 = 살아남음
   - 역사적 근거: 결백 주장한 19명 전원 교수형. 자백한 54명 전원 생존

2. **"저 아이는 연기하고 있어요" — 고발자 반박** ✅
   - 현대 논리: 명백한 히스테리 연기를 논리적으로 반박한다
   - 1692 실제: 고발자 반박 = 악마의 편 = 즉각 고발 대상 전환
   - 역사적 근거: 조지 버로스 목사는 논리적 반박 후 교수형

3. **"재판을 거부합니다" — 침묵/무응답** ⚠️ 단일 출처
   - 현대 논리: 불공정한 재판이니 거부할 수 있다
   - 1692 실제: 재판 거부 = Peine forte et dure(프레싱). 돌로 눌려 압사
   - 역사적 근거: Giles Corey 사례 (단, 그의 전략은 재산 보호 목적이었음)

4. **재산 증거 서류 제시 — 문서로 반박** ✅
   - 현대 논리: 법적 서류로 억울함을 증명한다
   - 1692 실제: 재판관이 피고인의 재산 규모를 파악하는 역효과. 재산이 클수록 표적이 됨
   - 역사적 근거: 재산 몰수 인센티브. 지주들이 집중 고발된 패턴

5. **"공범 이름을 댈 수 없습니다" — 거짓 자백 거부** ✅
   - 현대 논리: 없는 공범 이름을 댈 수 없다 (거짓말은 죄다)
   - 1692 실제: 공범 이름을 대지 않으면 자백 효과 반감. 처형 가능성 상승
   - 역사적 근거: 자백자들은 공동체 이웃을 공범으로 지목해 연쇄 고발 유발

6. **이웃/유력자에게 도움 요청 — 사회적 연대** ✅
   - 현대 논리: 지역 유지나 친분 있는 사람에게 구명 요청한다
   - 1692 실제: 유력자도 고발됨. 접촉 자체가 공범 의혹으로 연결 가능
   - 역사적 근거: Rebecca Nurse(교회 집사), Martha Corey(교회 신도) 등 지역 유지도 처형

## 5. 플레이어 포지션 후보

| 신분 | 취약점 | 강점 | 게임적 긴장감 |
|------|--------|------|--------------|
| 하녀 (집 안에 고발자 소녀 있음) | 집 안에서 목격자, 도망 불가 | 고발자 심리 근거리 관찰 가능 | 가장 높음 ★★★★★ |
| 지주 여성 (땅 소유) | 재산 = 표적, 재판관 이해관계 직결 | 문서/자원 보유 | 높음 ★★★★☆ |
| 감옥 이미 수감된 피의자 | 이미 기소 상태에서 시작 | 감옥 내 정보 네트워크 | 중간 ★★★☆☆ |

## 6. 에피소드 스탯 설계 후보

| 스탯 | 의미 | 오르는 행동 | 내려가는 행동 | Trade-off |
|------|------|------------|--------------|-----------|
| **신앙 (Faith)** | 공동체 내 종교적 신뢰도 | 예배 참석, 기도, 찬송 | 이성적 발언, 반박 | 신앙↑ = 평판↑ but 법적방어력↓ |
| **평판 (Reputation)** | 이웃 사이 신뢰도 | 이웃 돕기, 침묵 유지 | 고발자 반박, 낯선 행동 | |
| **법적방어력 (Defense)** | 재판에서 버틸 수 있는 힘 | 증거 확보, 증인 확보 | 스펙트럴 에비던스 적중 | Defense↑ 시도 = 표적 노출 위험 |
| **정신력 (Sanity)** | 극한 상황 버티는 힘 | 수면, 지지자 접촉 | 감옥 체류, 발작 목격 | 0이 되면 자백 |

**핵심 Trade-off**: 신앙을 높이면 커뮤니티 생존율은 오르지만 논리적 자기방어가 불가능해짐. 법적방어력을 높이려 하면 재산/지식이 드러나 표적이 됨.
