# EP01 유튜브 대본 집필 브리핑 (페이블 전달용)
*Sonnet이 정리, 실제 집필은 페이블 담당 — 2026-07-10*

## 담당 모델
Step 4 계열(창작 글쓰기)은 페이블 담당. 이 문서는 집필에 필요한 자료/결정사항 정리본.

## 확정된 루트 — final_B (자백), 정신력 > 50 분기
- stage1: **B** (예배 + 엿들음 + 서류 작성 → `land_deed_info` 획득, 사용하지 않음)
- stage2: 자유 선택 가능하나 **A/E는 피할 것** — E는 서류를 조기 제시해 압수당해 이 루트의 의미가 사라짐.
  A(발작 연기)나 C(결백 주장)를 추천. 단, 정신력을 50 초과로 유지해야 함(과다 감소 선택 피할 것).
- stage3: **A** (세라의 조언 수락 → `confession_strategy` 획득, "프루던스" 이름 확보)
- final: **B**, 그중 **정신력 > 50 분기만** — "프루던스" 한 명만 지목하고 끝남.
  **정신력 ≤ 50 분기(메리까지 지목하는 오염 버전)는 이번 영상에서 사용하지 않음.**

### 왜 이 루트인가 (사용자 결정 배경)
- 퍼펙트 엔딩(final_C, 서류로 무죄 증명)은 게임의 "정답"을 그대로 스포일링해서 게임 유인 효과를 죽임 — 제외.
- 단순 사망 엔딩(final_A)보다 자백 엔딩이 더 날카로운 반전: "정직하면 죽고 거짓말하면 산다"는 걸
  시청자가 직접 목격하게 하는 게, 그냥 죽는 것보다 교육적 충격이 큼 + 도덕적 찜찜함이 남아 댓글 유도에도 좋음.
- 정신력 > 50 분기를 쓰는 이유: 도덕적 타협을 "죽은 사람 이름 하나"로 깔끔하게 제한 — 오염 버전(산 사람도 팔아버림)까지
  넣으면 단일 영상에 감정 비트가 너무 많아짐.

## 형식
- 유튜브 보이스오버 대본 (`youtube_script.md`)
- **오르페아 개입을 게임 대본보다 훨씬 자주** — 게임은 "씬당 최대 1회, S3부터 유머 예산 0" 규칙이지만
  유튜브는 정반대. 오르페아가 거의 매 장면 해설·조롱·팩트 삽입을 해야 유튜브 나레이션으로서 기능함.
- 목표 분량: 7000자 이상 (agents/extractor.md 기준)

## 참조 순서
1. `references/orphea_voice_guide.md` — 오르페아 목소리 공식 (필수)
2. `vn_script.md` — 원본 대본. 위 루트(opening 전체 → stage1_B_01~02 → stage1_arrest → stage2_entry →
   [stage2 선택] → stage3_entry → stage3_01 → stage3_A → final_entry → final_touch/refuse → final_main →
   final_B → epilogue → game_over)를 따라가며 확장
3. `fact_sheet.md` — 각 반전 지점에서 실제 역사 팩트로 뒷받침 (예: 자백자 54명 전원 생존 vs 결백주장 19명 전원 교수형)
4. `scenario.json` — 스탯/아이템 정의, `core_rules`(정체 은폐 규칙 등)
5. `agents/extractor.md`의 "출력 1: youtube_script.md" 변환 규칙 — NPC 대사→오르페아 해설 확장,
   CHOICE 직전→"댓글에 뭘 골랐는지 남겨보세요", STAT 변화→보이스오버에서 자연스럽게 언급,
   final_B는 GOOD_END가 아니라 애매한 결말이므로 "에필로그 + 구독유도"와 "배드엔딩 팩트 폭로" 중간 톤으로 처리
