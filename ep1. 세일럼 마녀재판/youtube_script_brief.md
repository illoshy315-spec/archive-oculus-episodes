# EP01 유튜브 대본 집필 브리핑 (페이블 전달용)
*Sonnet이 정리, 실제 집필은 페이블 담당 — 2026-07-10*

## 담당 모델
Step 4 계열(창작 글쓰기)은 페이블 담당. 이 문서는 집필에 필요한 자료/결정사항 정리본.

## 확정된 루트 — final_B (자백), 정신력 > 50 분기 (2026-07-10 stage1 변경: B→A)
- stage1: **A** (탄원서 — 메리와 서명 수집 → 제출 → 옆 칸 기재. 평판 -30)
- stage2: **빵 먹기(eat)** → 심문에서 **A(발작 연기)**. 맥각 감염(ergot_infected)으로 "검은 형체" 모호성 비트 확보.
  E는 아이템 미보유라 자동 미표시, D(고발자 반박)는 즉사 루트라 제외. 정신력: 100−20(빵)−10(S3 부하 압박)=70 > 50 ✓
- stage3: **A** (세라의 조언 수락 → `confession_strategy` 획득, "프루던스" 이름 확보)
- final: 터치 테스트는 **touch(만진다)** — "설계상 무죄가 없는 검사" 팩트 비트. 이후 **B(자백)**,
  그중 **정신력 > 50 분기만** — "프루던스" 한 명만 지목하고 끝남.
  **정신력 ≤ 50 분기(메리까지 지목하는 오염 버전)는 이번 영상에서 사용하지 않음.**

### 왜 이 루트인가 (사용자 결정 배경)
- 퍼펙트 엔딩(final_C, 서류로 무죄 증명)은 게임의 "정답"을 그대로 스포일링해서 게임 유인 효과를 죽임 — 제외.
- **stage1_B(예배+서류)도 같은 이유로 제외 (2026-07-10 사용자 지적)**: 퍼펙트 루트의 길목이라 부분 스포일러 +
  자백 엔딩에선 서류를 안 쓰니 "안 쏘는 총"만 남음.
- **stage1_A(탄원서)를 쓰는 이유**: 이 에피소드의 대표 생존 지뢰(서명=증거=안전 → 공범 명단 반전) +
  메리의 법정 배신이 중반 드라마 비트 + "합리적 시도 전부 실패 → 거짓 자백으로 생존"이라는 서사 아치가
  자백 엔딩의 주제를 루트 자체로 증명함.
- 단순 사망 엔딩(final_A)보다 자백 엔딩이 더 날카로운 반전: "정직하면 죽고 거짓말하면 산다"는 걸
  시청자가 직접 목격하게 하는 게, 그냥 죽는 것보다 교육적 충격이 큼 + 도덕적 찜찜함이 남아 댓글 유도에도 좋음.
- 정신력 > 50 분기를 쓰는 이유: 도덕적 타협을 "죽은 사람 이름 하나"로 깔끔하게 제한 — 오염 버전(산 사람도 팔아버림)까지
  넣으면 단일 영상에 감정 비트가 너무 많아짐.

## 형식
- 유튜브 보이스오버 대본 (`youtube_script.md`)
- **오르페아 개입을 게임 대본보다 훨씬 자주** — 게임은 "씬당 최대 1회, S3부터 유머 예산 0" 규칙이지만
  유튜브는 정반대. 오르페아가 거의 매 장면 해설·조롱·팩트 삽입을 해야 유튜브 나레이션으로서 기능함.
- 목표 분량: 7000자 이상 (agents/extractor.md 기준)

## 시청 유지(리텐션) 팁 — 2026-07-10 추가
1. **오프닝 20~30초 안에 이 에피소드의 핵심 반전을 떡밥으로 던질 것**: "자백한 사람은 전원 생존, 결백 주장한
   19명은 전원 교수형" 같은 역사적 사실을 초반에 미리 티저로 깔아서 "어떻게 그게 가능하지?"를 궁금하게 만듦.
   (이건 "이번 회차가 어떤 선택을 하는가"의 스포일러가 아니라 일반 역사 사실 떡밥이라 무방함)
2. **패턴 인터럽트 밀도를 게임 대본보다 훨씬 높게**: 스탯변화/효과음/시스템 알림을 대사 3~4줄마다 한 번꼴로.
3. **생존확률 숫자를 처음 한 번만 던지지 말고 선택마다 계속 갱신**: 스코어보드처럼 오르내리는 걸 보는 재미.
4. **오르페아가 극중 "플레이어"가 아니라 화면 너머 실제 시청자에게 직접 묻는 순간을 선택 직전 1~2회 삽입**:
   "당신이라면 어떻게 했을까요?" 류 — 댓글 유도 효과.
5. **팩트 설명이 3문장 이상 이어지지 않게**: 사이사이 대사/스탯변화/오르페아 독설로 끊을 것.
6. **엔딩 CTA는 구체적으로**: "구독해주세요" 대신 "이 게임에서 프루던스 말고 다른 선택을 하면 어떻게 되는지
   직접 확인해보세요" 식으로 궁금증을 남겨서 게임 클릭을 유도.
7. **엔딩 CTA는 이중 링크 (2026-07-10 사용자 확정)**: ① 다른 선택지의 결말이 궁금한 사람 → 게임 링크,
   ② 역사적 사실의 전체 기록이 궁금한 사람 → orpheaarchive.com 아카이브 링크. 본편 중간에 걸어둔
   열린 고리 두 개("직접 확인하는 방법은 나중에" — 파트2 선택지 / "전체 기록은 영상 끝에" — 파트3 버로스)를
   여기서 각각 회수한다. 중간에는 링크를 절대 미리 주지 않는다.
