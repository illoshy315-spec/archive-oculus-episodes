# 챗 프롬프트 — EP01 씬 비트 설계
*이 파일 전체를 Claude.ai 챗에 붙여넣기*

## 태스크

Archive Oculus EP01 세일럼 마녀재판의 **씬 비트**를 설계해줘.

Archive Oculus는 역사 생존 시뮬레이션 비주얼 노벨이야.
플레이어가 역사 속 인물로 접속해서 "현대 상식으로는 정답처럼 보이지만 당시엔 사망"인 함정을 피하며 생존하는 게임.

씬 비트란: 대사를 쓰기 전에 "이 씬에서 무슨 일이 어떻게 일어나는가"를 설계하는 것.
대사 없이 감정/구조/NPC 충돌만 설계한다. 대사는 나중에 쓴다.

---

## 씬 비트 양식 (각 스테이지마다 채울 것)

```
[씬 이름]
감정 목표:        이 씬이 끝날 때 플레이어가 느껴야 할 감정 1개
압박 단계:        환경 압박 → 사회적 압박(NPC 충돌) → 내적 압박 순서로
NPC 충돌 구조:    누가 누구와 어떻게 충돌하는가 (플레이어 샌드위치 구조)
직접 말하지 않는 것: 이 씬에서 행동으로만 보여줄 정보 (대사로 말하면 안 되는 것)
선택지 직전 미해결 의문: 선택지가 나올 때 플레이어 머릿속에 떠있어야 할 질문 1개
복선 위치:        타입A(2스테이지 이상 앞)/B(직전)/C(엔딩 이후)/D(행동형) 중 하나
친교 씬:          있다면 — 어느 NPC의 어떤 예상 밖 모습 (3~5줄, 정보 전달 없음)
```

## 극작 원칙 (반드시 지킬 것)

**빌런 규칙 (역전재판 타쿠미 원칙):**
- 빌런은 자기 계획을 절대 직접 말하지 않는다
- 행동과 질문으로만 의도를 드러낸다
- 플레이어가 빌런의 의도를 스스로 계산하게 만든다

```
❌ 금지: "나는 당신의 토지를 빼앗을 것이다"
✅ 허용: 코윈이 지도를 펼친다. 손가락이 강변 쪽을 짚는다.
         "강변 쪽 땅 주인이 누구지?"
         대답을 듣는다. 손가락을 떼지 않는다.
```

**NPC 충돌 구조 (단간론파 코다카 원칙):**
- NPC들이 플레이어에게 독백하면 압박이 없다
- 두 NPC가 충돌할 때 플레이어는 그 사이에 끼여야 한다

```
❌ 금지: NPC A가 플레이어에게 고발 → NPC B가 플레이어를 변호
✅ 허용: NPC A가 NPC B를 가로막는다. NPC B의 눈이 흔들린다.
         플레이어는 그 틈에서 정보를 읽는다.
```

**복선 원칙 (우미네코 원칙):**
- 복선은 당시 맥락에서 완전히 자연스러워야 한다
- 인위적으로 튀는 복선은 복선이 아니라 예고편이다
- 나중에 회수될 때 "그게 그거였어!" 하는 순간이 생겨야 한다

**Show Don't Tell:**
| 금지 (직접 말함) | 허용 (행동으로 보임) |
|---|---|
| "저는 당신을 배신할 겁니다" | NPC가 플레이어 뒤에서 빌런 측과 눈을 맞춘다 |
| "이 물건이 중요합니다" | NPC가 물건을 건네며 아무 말도 하지 않는다 |
| "긴장됩니다" | 플레이어가 손을 쥐는 묘사 |

**선택지 직전 5단계 (역전재판+단간론파 공통):**
```
1. 공간 확립 — 어디에 누가 있는지
2. 균형 붕괴 — 기대하던 것과 다른 일이 벌어진다
3. NPC 반응 — 각자 다른 방향으로 반응 (충돌)
4. 압박 — 시간 압박 or 시선 압박
5. 선택지 등장
```

## 에피소드 정보

**플레이어**: 엘리자베스, 27세, 과부, 강변 농지 200에이커, 6살 아들
**핵심 취약점**: 재산 있는 혼자인 여성 = 고발 표적 1순위

**스탯**: 신앙(80) / 평판(70) / 법적방어력(30) / 정신력(100)

**아이템 2개:**
- `land_deed_info` (토지 양도 서류): S1-B에서 획득 → 최종관문 C에서 사용
- `confession_strategy` (세라의 조언): S3-A에서 획득 → 최종관문 B에서 사용

## NPC 4명

**아비게일 윌리엄스 (villain)**
- 등장: 오프닝, S1, 최종관문
- 표면: 악마에게 시달리는 11세 소녀
- 실제: 고발을 멈추면 본인도 의심받는 구조. 멈출 수 없다.
- 말투: 평소엔 순하고 조용. 발작 중엔 완전히 다른 사람.
- 버릇: 두 손 앞에 모으고 고개 숙임 / 발작 직전 눈 초점 잃음 0.5초
- 대사: "저기... 저 사람이요. 어젯밤에 제 방에 왔어요."

**조너선 코윈 (boss)**
- 등장: 최종관문 (S1 예배씬에서 목소리만)
- 표면: 신의 이름으로 마녀를 심판하는 재판관
- 실제: 플레이어의 토지 200에이커를 합법적으로 몰수하려 함
- 말투: 격식 있고 낮고 느림. 절대 소리 지르지 않음. "다만..."으로 조건을 붙임.
- 버릇: 지도를 손가락으로 훑음 / 발언 전 안경 고쳐 씀
- S3-C(뇌물) 선택 시: 등장 전에 이미 토지 규모 파악. 선제 압박.

**메리 (fake_ally)**
- 등장: S1, S3 진입 이벤트(눈 마주침), 최종관문
- 표면: 함께 싸우자는 따뜻한 이웃
- 실제: 자신이 살아남기 위해 플레이어를 방패로 쓰거나 먼저 고발할 준비가 됨
- 말투: 따뜻하고 빠름. "우리끼리만 아는 얘긴데..."로 시작.
- 버릇: 손을 많이 씀 / 신체 접촉 많음 / 법정에서 시선 바닥 고정
- S1-A 선택 시: 최종관문에서 공범 지목 증언 ("저 사람이 저와 함께 탄원서를 썼습니다")
- S1-A 미선택 시: 법정에서 "저는 잘 모르겠어요" 침묵 중립

**세라 (ally)**
- 등장: S3
- 표면: 차갑고 무서운 수감자
- 실제: 같은 자리를 먼저 지나온 생존자. 방법을 안다.
- 말투: 건조하고 짧음. 주어 생략. 감정 단어 없음.
- 버릇: 벽에 등 기대고 앉음 / 말할 때 상대 안 봄 / 쇠고랑 소리 줄이려 손목 조절
- S2-A(악마봤다) 선택 시: 첫 대사 "당신 이미 한 번 팔았군요. 그럼 방법은 알겠네요."
- 정보 전달 시: "자백해. 근데 이름은 하나만 대. 이미 죽은 사람 이름으로."

## 스테이지 구조

### 오프닝
선택지 없음. 아비게일 발작 현장. 오르페아 등장. 신분 부여.
오르페아: "축하드려요. 1692년 세일럼에 접속하셨습니다. 생존 확률 23%."

### 스테이지 1 — 공동체 / 젠더 / 정보·소문
함정: A (탄원서 — 메리와 함께 아비게일 고발)
탈출구: B (예배 참석 → 코윈 엿들음 → land_deed_info 획득)
웃긴: C (이웃집 고양이를 마녀 사역자로 고발 — 당시엔 정상적 행동)

역사적 근거:
- A 함정: 탄원서는 법적 효력 없음. 서명자가 공범 목록이 됨.
- B 탈출구: Giles Corey가 실제로 재판 전 아들에게 재산 양도해 몰수 막음.
- C 웃긴: 티투바 증언에 "붉은 고양이, 검은 개"가 악마의 사역자로 법적 채택됨.

### 스테이지 2 — 법/제도 / 종교 / 과학·의학
진입 이벤트: 맥각균 오염 빵 (먹는다 → 정신력-20+ergot_infected 플래그 / 참는다 → 정신력-10)
함정: C ("저는 결백합니다" → 법적방어력-40, 발언권 박탈)
리스크: A ("저도 악마를 봤습니다" → 신앙+30, S3 공범 압박 이벤트 발생)
소극: B (침묵 → 정신력-20)

역사적 근거:
- C 함정: 결백 주장한 19명 전원 교수형. 자백한 54명 전원 생존.

### 스테이지 3 — 경제 / 공동체 (감옥)
진입 이벤트: 메리와 눈 마주침 (S1-A 선택 시 메리가 시선 피함 / 미선택 시 고개 끄덕임)
조건부 이벤트: S2-A 선택 시 심문관 부하가 "공범 이름 세 명 더 대시오" 압박
탈출구: A (세라 말 듣기 → confession_strategy 획득)
함정: B (거짓 자백 거부 → confession_strategy 소실, 최종관문 B 비활성)
리스크: C (간수 뇌물 → 코윈에게 재산 정보 노출, 최종관문 선제 압박)

### 최종관문 — 법정
함정: A ("결백합니다. 물리적 증거를 요구합니다" → 교수형)
탈출구 B: confession_strategy 필요 ("자백하되 이름은 하나만")
탈출구 C: land_deed_info 필요 ("토지는 이미 양도했습니다")

## 씬 비트 설계 요청

위 내용을 바탕으로 아래 각 씬의 씬 비트를 설계해줘.

1. **오프닝** — 아비게일 발작 현장 + 오르페아 등장 + 신분 부여
2. **S1 진입 씬** — 메리 등장. 선택지 직전까지.
3. **S1-A 분기** — 탄원서 제출 후
4. **S1-B 분기** — 예배 참석 → 코윈 엿들음 → land_deed_info 획득
5. **S1-C 분기** — 고양이 고발
6. **S2 진입 이벤트** — 맥각균 빵
7. **S2 심문 씬** — 선택지 직전까지
8. **S3 진입 이벤트** — 메리 눈 마주침
9. **S3 감옥 씬** — 세라 등장 + 선택지까지
10. **최종관문** — 법정 씬

각 씬 비트마다 양식 채워줘. 대사는 쓰지 말고 구조만.
