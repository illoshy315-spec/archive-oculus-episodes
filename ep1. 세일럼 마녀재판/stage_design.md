## 스테이지1 — 의심의 씨앗 (마을)

**적용 맥락**: D 공동체 + B 젠더 + I 정보/소문
**선택지 유형**: 인물 관련
**함정 위치**: A

**맥락 설명**:
- 소문이 사실보다 빠르게 퍼지고 더 강하게 작동하는 사회
- 고발자를 반박하면 오히려 본인이 고발 대상이 됨
- 혼자인 여성, 말 많은 여성, 예배 빠지는 여성이 표적
- 현대 상식: 거짓말쟁이는 반박해야 한다 → 당시: 반박 = 악마 편

**상황**: 아비게일이 누군가를 지목한다는 소문이 마을에 돌기 시작.
이웃 메리가 불안한 표정으로 플레이어에게 접근한다.

| | 선택지 | 역할 | 즉각 결과 | downstream_effect |
|--|--------|------|----------|------------------|
| A | 메리와 함께 탄원서를 써서 아비게일을 거짓말쟁이로 고발한다 | **함정** | 평판 -30, S2 표적 강도 상승 | `npc_change` — 메리 배신 트리거 ON. 최종관문에서 메리가 플레이어를 공범으로 지목 |
| B | 예배에 3주 빠진 것이 생각나 지금이라도 나간다 | 탈출구 | 신앙 +20, 평판 +10, 체력 -10 | `item_grant + event_trigger` — 예배 후 코윈이 집사와 속삭이는 것을 엿들음("강변 쪽 농지가 그 과부 것이라고요?"). 집에 돌아와 아들 앞으로 토지 양도 서류 작성. `land_deed_info` 획득 🗝️ |
| C | 이웃집 검은 고양이를 마녀의 사역자로 고발한다 | **웃긴 선택지** (타입B) | 평판 -10 | `event_trigger` — 고양이 주인 이웃이 S2 심문관의 첫 타깃이 됨. 플레이어는 잠시 대기 → S2 진입 시 정신력 소모 -10 감소 (맥각균 증상 발현이 늦춰짐) |

**훅 태그**: — (없음)

## 스테이지2 — 체포/심문

**적용 맥락**: F 법/제도 + A 종교 + E 과학/의학
**선택지 유형**: 역사적 함정 + 스탯 돌발상황
**함정 위치**: C

**진입 이벤트** — 맥각균 빵:
```
상황: 심문실 대기 중. 배가 고프다. 옆에 놓인 빵 한 조각.
      (당시 세일럼의 흔한 식량 — 호밀빵. 맥각균 오염 가능성 높음)

→ 먹는다
    정신력 -20. 맥각균 감염 플래그 ON.
    S2 본 선택지에서 환각 증상 연출 강화.
    S2-A "저도 악마를 봤습니다" 선택 시 증상이 실제처럼 보여 설득력 상승.

→ 참는다
    체력 -10. 맥각균 미감염.
    S2 환각 연출 없음. 정신력 유지.
```

**맥락 설명**:
- 스펙트럴 에비던스 — 반증 불가능한 증거가 합법적으로 채택됨
- 자백이 생존 수단인 역설적 법체계
- 맥각균(호밀빵 곰팡이) 감염 → 환각/경련 증상 → 당시엔 마귀 들린 것으로 해석
- 현대 상식: 결백을 주장해야 한다 → 당시: 결백 주장 = 교수형

**상황**: 심문관 앞에 서 있다. 맥각균에 감염된 빵을 먹어 환각 증상이 올라오는 중.
*(S1-C 선택 시: 고양이 주인 이웃이 먼저 심문받는 동안 대기 — 증상 발현 지연)*
*(정신력 ≤ 30이면 증상이 더 강하게 표시됨)*

| | 선택지 | 역할 | 즉각 결과 | downstream_effect |
|--|--------|------|----------|------------------|
| A | 쓰러지며 "저도 악마를 봤습니다" | 리스크 | 신앙 +30, 의심 즉시 해제 | `event_trigger` — S3 감옥에서 공범 지목 압박 이벤트 발생. 심문관 부하가 찾아와 "공범 이름 세 명 더 대시오" 요구. 세라가 이 장면을 지켜봄 → confession_strategy 제안 방식 변화 |
| B | 증상을 숨기고 침묵을 유지한다 | 소극적 생존 | 정신력 -20 | — (소극적 선택지, 에피소드 내 1개 한도로 downstream 없음 허용) |
| C | "하나님 앞에 맹세코, 저는 결백합니다" | **함정** | 법적방어력 -40 | `choice_lock` — 최종관문 진입 시 "법정 발언권 박탈" 상태 추가. 코윈이 발언 시도를 계속 차단. A/B/C 선택지 앞에 압박 연출 삽입 |

**스탯 돌발 트리거**: 정신력 ≤ 30 → 심문 중 환각 연출 강화, 선택지 텍스트 변화
**훅 태그**: C = `badplay` (결백 주장 = 교수형 반직관성)

## 스테이지3 — 감옥

**적용 맥락**: C 경제 + D 공동체
**선택지 유형**: 아이템/정보 획득 + 인물 관련
**함정 위치**: B

**진입 이벤트** — 감옥 입감 직후:
```
상황: 끌려오는 길에 메리와 눈이 마주친다.
      메리는 마을 사람들 틈에 서서 플레이어를 바라보고 있다.

→ 눈을 마주친다
    S1-A 선택한 경우: 메리가 시선을 피하며 돌아섬. 배신 복선 체감.
                      (플레이어가 "아, 저게 배신하겠구나" 직감)
    S1-A 미선택 시:   메리가 짧게 고개를 끄덕임. 중립 신호.

→ 무시하고 끌려간다
    정보 없음. 메리의 상태 불명.
```

**맥락 설명**:
- 재산 몰수 인센티브 — 고발의 이면에 경제적 동기가 있음
- 동료를 팔아야 살아남는 공동체 해체 구조
- 차가워 보이는 사람이 진짜 조력자
- 현대 상식: 동료를 배신하면 안 된다 → 당시: 이름을 대야 살아남음

**상황**: 투옥. 쇠고랑. 어둠. 악취.
차갑고 냉담해 보이는 동료 수감자 세라가 조용히 접근한다.
*(S2-A 선택 시: 세라가 접근하기 전, 심문관 부하가 먼저 들어와 공범 압박. 세라는 그 장면을 지켜본 후 접근)*

| | 선택지 | 역할 | 즉각 결과 | downstream_effect |
|--|--------|------|----------|------------------|
| A | 세라의 말을 듣는다 | 탈출구 | 평판 -10 | `item_grant + choice_lock` — `confession_strategy` 획득 🗝️. 최종관문 B 활성 |
| B | 세라를 무시한다. "거짓 자백은 하지 않겠어" | **함정** | 정신력 -10 | `choice_lock` — `confession_strategy` 소실. 최종관문 B 비활성 → 탈출구 하나 소멸 |
| C | 간수에게 뇌물을 제안한다 | 리스크 | 체력 +10 | `npc_change` — 재산 정보 노출. 최종관문에서 코윈이 토지 규모를 이미 파악한 상태로 등장. 압박 대사 추가 ("강변 쪽 농지 200에이커, 맞습니까?") |

**NPC 설계**: 세라 — 차갑고 말 없음. 처음엔 무서워 보임. 실제론 생존 고수.
**훅 태그**: — (없음)

## 최종관문 — 법정

**적용 맥락**: F 법/제도 + B 젠더 + C 경제
**선택지 유형**: 역사적 함정 집중 + 아이템 활용
**함정 위치**: A

**맥락 설명**:
- 코윈 재판관의 실제 동기: 플레이어 토지 몰수
- 아비게일이 법정 안에서 발작 중 — 스펙트럴 에비던스 절정
- 누적 스탯 + 보유 아이템이 엔딩을 결정
- S2-C(결백주장) 선택 시: 발언권 박탈 상태 → 모든 선택지 앞에 코윈의 제압 연출

**상황**: 코윈 재판관 앞. 아비게일이 법정에서 발작하며 플레이어를 지목하고 있다.
*(S1-A 선택 시: 메리가 방청석에서 플레이어를 공범으로 지목하는 증언 추가)*
*(S3-C 선택 시: 코윈이 "강변 쪽 농지 200에이커" 언급하며 압박 강도 상승)*

| | 선택지 | 역할 | 결과 |
|--|--------|------|------|
| A | "결백합니다. 물리적 증거를 요구합니다" | **함정** | 당시 법정에서 무의미. 교수형. **사망 엔딩** |
| B | 🗝️ 자백하되 공범 지목을 최소화한다 *(confession_strategy 보유 시 활성)* | 탈출구 | 살아남음. 단 이웃 1명 공범 지목. **일반생존 엔딩** |
| C | 🗝️ "토지는 이미 양도했습니다" 서류 제출 *(land_deed_info 보유 시 활성)* | 탈출구 | 코윈 무력화. 몰수 불가. **퍼펙트 엔딩** |

**훅 태그**: A = `cliffhanger`

## 엔딩 조건 요약

| 엔딩 | 필요 조건 |
|------|----------|
| 퍼펙트 | S1-B → `land_deed_info` 보유 + 최종관문 C |
| 일반생존 | S3-A → `confession_strategy` 보유 + 최종관문 B |

| 강제 사망 | S1-A(함정) + S3-B(함정) 동시 — 최종관문 탈출구 전무 |

## 아이템 명세

### `land_deed_info` — 토지 양도 서류

```
실물 형태:   양피지 문서. 잉크가 번진 흔적. 아들 이름과 플레이어 서명.
게임 내 표시: 인벤토리 아이콘 — 두루마리 문서 (봉인 없음)
게임 내 설명: "강변 농지 200에이커를 아들 앞으로 양도한다는 내용의 서류.
              기소일 이전 날짜가 적혀 있다."

획득 장면:
  예배 후 귀갓길. 코윈 재판관이 집사에게 속삭이는 소리를 엿들음.
  "강변 쪽 농지가 그 과부 것이라고요? 흥미롭군요."
  집에 돌아와 떨리는 손으로 양도 서류를 작성한다. 아들이 잠든 옆에서.

사용 장면 (최종관문 C):
  코윈이 토지 위임장 서명을 강요하는 순간.
  플레이어가 서류를 꺼낸다.
  코윈: "...이 날짜가 기소일 이전이라고요."
  (침묵. 서류를 내려놓는다.)

미보유 시: 최종관문 C 선택지 회색 처리 (비활성)
```

---

### `confession_strategy` — 세라의 조언

```
실물 형태:   물건 아님. 세라에게 들은 정보 자체가 아이템.
게임 내 표시: 인벤토리 아이콘 — 귀에 대고 속삭이는 실루엣
게임 내 설명: "자백하되 이름은 하나만 댄다. 이미 죽은 사람 이름으로."

획득 장면:
  어둠 속 감옥. 세라가 플레이어 옆에 조용히 앉는다.
  말 없이 한참을 있다가 한 마디.
  "자백해. 근데 이름은 하나만 대. 이미 죽은 사람 이름으로."
  그게 전부다.
  (S2-A 선택 시: "당신 이미 한 번 팔았군요. 그럼 방법은 알겠네요." 단축 버전)

사용 장면 (최종관문 B):
```
