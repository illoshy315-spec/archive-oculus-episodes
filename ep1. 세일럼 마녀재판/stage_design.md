# EP.01 세일럼 마녀재판 — 스테이지 설계
*Step 3 산출물 — 파트 구조 + 선택지 A/B/C + 사회문화 맥락*

---

## 파트 구조 확정

```
포맷: 기본 생존형
스테이지 수: 3개
엔딩: 퍼펙트 / 일반생존 / 사망 (히든 없음)

오프닝       → 1692년 세일럼 빌리지. 아비게일 발작 현장. 오르페아 등장.
스테이지1    → 의심의 씨앗 (마을)
스테이지2    → 체포/심문
스테이지3    → 감옥
최종관문     → 법정
엔딩
```

---

## 오프닝

선택지 없음. 세계관 진입.

- 1692년 세일럼 빌리지, 새벽
- 아비게일 윌리엄스의 발작 현장에서 시작
- 오르페아 첫 등장 + 신분/상황 부여
- 스탯 초기값 설정

**오르페아 오프닝 멘트 방향**:
"축하드려요. 1692년 세일럼에 접속하셨습니다. 생존 확률 23%. 역사적으로 인기 있는 자살 루트입니다."

---

## 스테이지1 — 의심의 씨앗 (마을)

**적용 맥락**: D 공동체 + B 젠더 + I 정보/소문
**선택지 유형**: 인물 관련

**맥락 설명**:
- 소문이 사실보다 빠르게 퍼지고 더 강하게 작동하는 사회
- 고발자를 반박하면 오히려 본인이 고발 대상이 됨
- 혼자인 여성, 말 많은 여성, 예배 빠지는 여성이 표적
- 현대 상식: 거짓말쟁이는 반박해야 한다 → 당시: 반박 = 악마 편

**상황**: 아비게일이 누군가를 지목한다는 소문이 마을에 돌기 시작.
이웃 메리가 불안한 표정으로 플레이어에게 접근한다.

| | 선택지 | 맥락 | 결과 |
|--|--------|------|------|
| A | 메리와 함께 탄원서를 써서 아비게일을 거짓말쟁이로 고발한다 | I 정보/소문 **함정** | 고발자 반박한 사람들이 역사적으로 오히려 고발당함. 평판 -30, 스테이지2 표적 강도 상승 |
| B | 예배에 3주 빠진 것이 생각나 지금이라도 나간다 | A 종교 — 당시 논리 정답 | 신앙 +20, 평판 +10 / 체력 -10. 예배에서 토지 서류 관련 정보 입수 가능 🗝️ |
| C | 집에서 조용히 지낸다 | D 공동체 — 소극적 생존 | 스탯 변화 없음. 정보 소실 → 스테이지2 진입 시 불리 |

**아이템 심기**: B 선택 시 `land_deed_info` 획득 (최종관문 퍼펙트 엔딩 조건)
**NPC 복선**: 메리 — 달콤해 보이는 이웃. 최종관문에서 배신 여부는 스탯에 따라 결정

## 스테이지2 — 체포/심문

**적용 맥락**: F 법/제도 + A 종교 + E 과학/의학
**선택지 유형**: 역사적 함정 + 스탯 돌발상황

**맥락 설명**:
- 스펙트럴 에비던스 — 반증 불가능한 증거가 합법적으로 채택됨
- 자백이 생존 수단인 역설적 법체계
- 맥각균(호밀빵 곰팡이) 감염 → 환각/경련 증상 → 당시엔 마귀 들린 것으로 해석
- 현대 상식: 결백을 주장해야 한다 → 당시: 결백 주장 = 교수형

**상황**: 심문관 앞에 서 있다. 맥각균에 감염된 빵을 먹어 환각 증상이 올라오는 중.
*(정신력 ≤ 30이면 증상이 더 강하게 표시됨)*

| | 선택지 | 맥락 | 결과 |
|--|--------|------|------|
| A | "하나님 앞에 맹세코, 저는 결백합니다" | F 법/제도 **역사적 함정** | 결백 주장한 19명 전원 교수형. 법적방어력 -40 → 사망 엔딩 경로 |
| B | 증상을 숨기고 침묵을 유지한다 | 소극적 생존 | 정신력 -20. 의심은 안 받음 → 스테이지3 진행 |
| C | 쓰러지며 "저도 악마를 봤습니다" | E 과학/의학 역이용 | 고발자와 같은 증상 패턴 → 의심 즉시 해제. 신앙 +30. 단 공범 지목 압박 시작 |

**스탯 돌발 트리거**: 정신력 ≤ 30 → 심문 중 환각 증상 연출 강화 (선택지 텍스트 변화)

## 스테이지3 — 감옥

**적용 맥락**: C 경제 + D 공동체
**선택지 유형**: 아이템/정보 획득 + 인물 관련

**맥락 설명**:
- 재산 몰수 인센티브 — 고발의 이면에 경제적 동기가 있음
- 동료를 팔아야 살아남는 공동체 해체 구조
- 차가워 보이는 사람이 진짜 조력자
- 현대 상식: 동료를 배신하면 안 된다 → 당시: 이름을 대야 살아남음

**상황**: 투옥. 쇠고랑. 어둠. 악취.
차갑고 냉담해 보이는 동료 수감자 세라가 조용히 접근한다. *(진짜 조력자)*

| | 선택지 | 맥락 | 결과 |
|--|--------|------|------|
| A | 세라를 무시한다. "거짓 자백은 하지 않겠어" | D 공동체 **함정** | `confession_strategy` 소실. 최종관문 B 선택지 비활성 → 탈출구 없음 |
| B | 세라의 말을 듣는다 | C 경제 — 당시 논리 정답 | `confession_strategy` 획득 🗝️. 평판 -10. 최종관문 B 활성 |
| C | 간수에게 뇌물을 제안한다 | C 경제 | 체력 +10. 단 재산 정보 노출 → 최종관문 코윈 압박 강도 상승 |

**NPC 설계**: 세라 — 차갑고 말 없음. 처음엔 무서워 보임. 실제론 생존 고수.

---

## 최종관문 — 법정

**적용 맥락**: F 법/제도 + B 젠더 + C 경제
**선택지 유형**: 역사적 함정 집중 + 아이템 활용

**맥락 설명**:
- 코윈 재판관의 실제 동기: 플레이어 토지 몰수
- 아비게일이 법정 안에서 발작 중 — 스펙트럴 에비던스 절정
- 누적 스탯 + 보유 아이템이 엔딩을 결정

**상황**: 코윈 재판관 앞. 아비게일이 법정에서 발작하며 플레이어를 지목하고 있다.

| | 선택지 | 맥락 | 결과 |
|--|--------|------|------|
| A | "결백합니다. 물리적 증거를 요구합니다" | F 법/제도 **역사적 함정** | 당시 법정에서 무의미. 교수형. **사망 엔딩** |
| B | 🗝️ 자백하되 공범 지목을 최소화한다 *(confession_strategy 보유 시 활성)* | 세라 아이템 활용 | 살아남음. 단 이웃 1명 공범 지목. **일반생존 엔딩** |
| C | 🗝️ "토지는 이미 양도했습니다" 서류 제출 *(land_deed_info + 스테이지1 B 선택 시 활성)* | C 경제 역이용 | 코윈 무력화. 몰수 불가. **퍼펙트 엔딩** |

---

## 엔딩 조건 요약

| 엔딩 | 조건 |
|------|------|
| 퍼펙트 | 스테이지1 B 선택 → `land_deed_info` 보유 + 최종관문 C 선택 |
| 일반생존 | 스테이지3 B 선택 → `confession_strategy` 보유 + 최종관문 B 선택 |
| 사망 | 최종관문 A 선택 OR 특정 스탯 0 도달 OR 누적 지뢰 과다 적중 |

## 아이템 인과관계 요약

| 아이템 | 획득 조건 | 회수 파트 | 효과 |
|--------|----------|----------|------|
| `land_deed_info` | 스테이지1 B (예배 참석) | 최종관문 C | 코윈 무력화, 퍼펙트 엔딩 |
| `confession_strategy` | 스테이지3 B (세라 수용) | 최종관문 B | 자백 최적화, 일반생존 엔딩 |

## 스탯 설계 (확정)

| 스탯 | 초기값 | 0이 되면 |
|------|--------|---------|
| 신앙 (Faith) | 80 | 공동체 추방, 표적 가속 |
| 평판 (Reputation) | 70 | 이웃 전원 적대, NPC 조력 불가 |
| 법적방어력 (Defense) | 30 | 심문/법정에서 선택지 제한 |
| 정신력 (Sanity) | 100 | 자백 강요 선택지 강제 발동 |
