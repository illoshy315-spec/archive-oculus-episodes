# EP01 씬 비트 — 세일럼 마녀재판
*Step 4 대본 집필 전 씬 구조 설계서*

---

## 클코 집필 지침

이 파일은 대본을 쓰기 전 씬의 뼈대를 정의한다.
각 항목을 읽고, 대본에서 **행동과 대사로 구현**하라.
항목 문장을 대본에 직접 인용하지 않는다.
살을 붙이되, 항목에 명시된 것은 반드시 포함한다.

**항목별 주의:**
- `선택지 직전 의문` = 설계 목표. 이 문장을 NARR/THINK에 그대로 쓰지 말 것. 앞선 행동들로 플레이어가 스스로 이 질문을 품게 만들어야 한다.
- `복선 위치` = 심는 위치. 복선임을 강조하거나 NARR로 설명하지 말 것. 배경처럼 묻혀야 한다.
- `직접 말하지 않는 것` = 절대 대사/NARR로 설명 금지. 행동·표정·소리로만.

- 집필 수칙: `references/harness.md`

## 씬 비트 양식 설명

```
감정 목표:          이 씬이 끝날 때 플레이어가 느껴야 할 감정 1개
압박 단계:          환경 → 사회(NPC 충돌) → 내적 순서
NPC 충돌 구조:      누가 누구와 어떻게 충돌. 플레이어는 그 사이에 낌
직접 말하지 않는 것: 대사로 설명하면 안 되는 것. 행동으로만 보여줄 것
선택지 직전 의문:   선택지 등장 시 플레이어 머릿속에 떠있어야 할 질문 1개
복선 위치:          A(2스테이지 이상 앞) / B(직전) / C(엔딩 이후) / D(행동형)
친교 씬:            NPC의 예상 밖 모습. 정보 전달 없음. 입체성만.
```

## 1. 오프닝 — 아비게일 발작 + 오르페아 등장 + 신분 부여

```
감정 목표:          불안 속 호기심. 게임인지 현실인지 모르는 감각적 혼란

압박 단계:          환경 → 예배당. 연기와 땀 냄새. 아비게일 발작 중. 군중이 공간을 조임
                    사회 → 군중 시선이 플레이어 쪽으로 쏠리기 시작
                    내적 → 오르페아 등장. 이 혼란 속 플레이어에게만 들리는 목소리

NPC 충돌 구조:      군중(익명 압박) ↔ 아비게일(발작 중심) 사이에 플레이어
                    아비게일은 플레이어를 직접 지목하지 않음
                    손가락이 허공을 가리킴. 그 방향에 플레이어가 있을 뿐

직접 말하지 않는 것: 재산 있는 혼자인 여성의 위험성
                     → 다른 여성들이 남편 곁으로 바짝 붙는 것으로 보여줌
                     아비게일이 연기라는 것
                     → 발작 직전 눈 초점 0.5초 소실
                     플레이어의 취약성
                     → 오르페아 신분 부여 시 "6살 아드님이 마당에서 기다리십니다"

선택지 직전 의문:   "저 소녀는 날 보고 있는 건가, 아닌가?"

복선 위치:          타입D — 아비게일 손가락이 허공 가리킬 때
                    그 방향 벽에 코윈 코트 자락이 스침. 발각 안 됨

친교 씬:            없음
```

## 2. S1 진입 — 메리 등장 / 선택지 직전

```
감정 목표:          온기와 안도. 그런데 뭔가 빠르다는 찜찜함

압박 단계:          환경 → 예배당 밖. 군중 흩어짐. 아들이 마당 돌담 곁에 있음
                           플레이어 긴장이 풀리기 시작하는 순간
                    사회 → 바로 그 순간 메리가 팔을 잡음. 주변 두리번거리며 속삭임
                           이웃 남성이 지나가며 메리를 흘겨봄 → 메리 0.5초 굳음 → 다시 웃음
                    내적 → 메리의 말이 모두 맞다. 그래서 더 무섭다

NPC 충돌 구조:      메리(접근, 공감) ↔ 마을 여론(이웃 남성 시선) 사이에 플레이어
                    플레이어는 메리가 굳었다 풀리는 0.5초를 목격함

직접 말하지 않는 것: 메리가 두려워하고 있다는 것
                     → 말할 때 손이 너무 많이 움직임
                     탄원서가 법적 효력이 없다는 것
                     → 메리가 "효과 있을 거야" 말할 때 눈을 맞추지 않음
                     메리가 재산에 관심 있다는 것
                     → 대화 중 시선이 강변 방향으로 1회 이탈. 말 없음

선택지 직전 의문:   "메리가 방금 강변 쪽을 왜 봤지?"

복선 위치:          타입B — 메리의 강변 시선. 이 씬에서 '강변'은 아무도 말하지 않음

친교 씬:            메리가 아들에게 말 없이 건빵 건넴. 아이를 보지 않음. 말도 없음
                    건네는 손과 받는 손이 잠깐 겹침. 메리는 이미 다른 곳을 보고 있음
```

## 3. S1-A 분기 — 탄원서 제출 후

```
감정 목표:          뭔가를 해냈다는 짧은 안도. 즉시 오는 서늘함

압박 단계:          환경 → 집무실 문 앞. 서기가 서류를 받아 목록에 기재
                           플레이어는 그 목록을 볼 수 없음
                    사회 → 메리가 팔을 꽉 쥠. 문 너머 코윈 목소리 들림
                           메리가 즉시 손을 뗌
                    내적 → 서기의 기재가 멈추지 않음
                           서명한 사람 수가 늘수록 목록이 길어짐

NPC 충돌 구조:      메리(공범 제안자) ↔ 코윈(문 너머 목소리, 보이지 않는 관찰자)
                    코윈은 등장하지 않음. 목소리만
                    메리는 코윈 목소리를 듣자마자 플레이어 팔에서 손을 뗌
                    플레이어는 자신의 이름이 방금 목록에 올라갔음을 깨달음

직접 말하지 않는 것: 서명이 공범 목록이 된다는 것
                     → 서기가 '탄원인'이 아닌 다른 칸에 이름을 적음
                     메리가 이미 결과를 알고 있다는 것
                     → 메리가 놀라지 않음

선택지 직전 의문:   없음 (결과 씬)

복선 위치:          타입A — 최종관문에서 메리가 증인석에 설 때
                    이 씬에서 메리가 팔을 뗀 동작이 회수됨

친교 씬:            없음
```

## 4. S1-B 분기 — 예배 후 코윈 엿들음 / land_deed_info 획득

```
감정 목표:          금지된 것을 본 자의 심장 박동. 정보를 얻었는데 무섭다

압박 단계:          환경 → 예배 후 복도. 사람들 빠져나감. 플레이어만 늦음
                           창문 너머 코윈 목소리 + 지도 펼쳐지는 소리
                    사회 → 플레이어 멈춤. 반대편에서 성도가 걸어옴
                           코윈 목소리 계속. 이동도 못 하고 들을 수도 없는 상황
                    내적 → 코윈: 손가락이 지도를 두드리는 소리. "강변 쪽."

NPC 충돌 구조:      코윈(보이지 않는 위협) ↔ 성도(노출 위협) 사이에 플레이어
                    코윈은 플레이어를 모름. 성도는 플레이어를 봄. 눈이 마주침
                    성도가 그냥 지나감. 플레이어는 숨을 참음

직접 말하지 않는 것: 코윈의 의도
                     → 손가락 두드리는 소리만. "강변 쪽 주인이 누구지?" 침묵
                     land_deed_info의 가치
                     → 아이템이 인벤토리에 조용히 들어옴. 오르페아 코멘트 없음

선택지 직전 의문:   없음 (정보 획득 씬)
                    플레이어에게 남는 것: "코윈은 내 땅을 알고 있다"

복선 위치:          타입A — 최종관문 C에서 코윈이 토지 서류 앞에 처음으로 말을 잃을 때
                    이 씬의 손가락 두드리는 소리가 회수됨

친교 씬:            없음
```

## 5. S1-C 분기 — 고양이 고발

```
감정 목표:          황당함. 그런데 아무도 웃지 않는다는 공포

압박 단계:          환경 → 이웃집 검은 고양이. 마을 사람이 성호를 그음
                    사회 → 플레이어 고발 → 군중 진지하게 고개 끄덕임
                           아무도 웃지 않음. 서기가 받아 적음. 너무 자연스럽게
                    내적 → 이게 맞는 세계다. 지금 여기서는. 그게 더 이상하다

NPC 충돌 구조:      마을 공동체(집단 동조) ↔ 고양이 주인(말하려다 눌림)
                    고양이 주인이 말하려 함 → 주변 시선에 눌려 닫힘
                    플레이어는 그가 닫히는 것을 목격

직접 말하지 않는 것: 이것이 실제로 일어난 일이라는 것
                     → 서기가 너무 자연스럽게 받아 적음
                     이 선택이 플레이어를 보호하지 못한다는 것
                     → 아비게일이 창문 너머에서 이쪽을 바라봄. 표정 없음

선택지 직전 의문:   없음 (결과 씬)

복선 위치:          타입C — 엔딩 에필로그에서
                    "이후 이 마을에서 13마리의 동물이 마녀 혐의로 처형되었다"
                    이 고양이가 그중 하나

친교 씬:            고양이 주인이 플레이어와 마주침
                    원망 없이 하늘을 봄. 한참 후 들어감
                    플레이어는 그 등을 봄
```

## 6. S2 진입 이벤트 — 맥각균 빵

```
감정 목표:          사소한 것이 사소하지 않을 수 있다는 인지. 뒤늦은 감각

압박 단계:          환경 → 심문 대기 공간. 옆 수감자가 빵 공유 제안
                           시큼하고 눅눅한 냄새
                    사회 → 방 안쪽 심문관 부하가 말 없이 지켜봄
                           거절하면 이상한 사람이 되는 구조
                    내적 → 먹어도 이상하고 안 먹어도 이상함
                           선택지 오기 전에 이미 선택이 강요됨

NPC 충돌 구조:      빵을 주는 수감자(호의) ↔ 심문관 부하(관찰자) 사이에 플레이어
                    부하는 아무 말 없음. 기록하는 것처럼 보일 뿐
                    부하 시선이 잠깐 플레이어 쪽으로 내려옴

직접 말하지 않는 것: 맥각균이 발작 증상을 유발한다는 것
                     → 빵을 준 수감자의 손이 가끔 떨림. 설명 없음
                     먹은 것이 기록될 수 있다는 것
                     → 부하의 시선이 잠깐 내려옴

선택지 직전 의문:   "저 부하는 내가 빵을 먹는 것을 봤나?"

복선 위치:          타입D — 이후 플레이어 발작 증상 묘사 시
                    이 빵의 시큼한 냄새가 회수됨

친교 씬:            없음
```

## 7. S2 심문 씬 — 선택지 직전

```
감정 목표:          말을 해도 지고 안 해도 지는 구조임을 체감하는 공포

압박 단계:          환경 → 심문실. 창문 없음. 테이블 하나. 양초 하나
                           밀폐된 땀과 가죽 냄새
                    사회 → 심문관이 말하지 않고 기다림. 침묵이 먼저 발언하게 만듦
                           보조가 플레이어가 말하기 전부터 기재 시작
                    내적 → 뭘 말해도 적힌다. 안 말해도 적힌다

NPC 충돌 구조:      심문관(침묵 압박) ↔ 보조(이미 결론이 있는 것처럼 기재)
                    플레이어는 둘 사이 테이블에 앉아 있음
                    심문관은 한 번도 위협하지 않음. 기다리는 것이 위협
                    보조가 한 번 고개 들어 플레이어를 봄 → 즉시 내려감

직접 말하지 않는 것: 결백 주장이 사형 선고라는 것
                     → 테이블 위 서류 맨 아래 칸에
                       '결백 주장 — 처리 완료' 서식이 이미 인쇄되어 있음
                       가까이 가야 보임
                     자백한 54명이 살았다는 것
                     → 보조 서류 옆 이름 목록. 이름들에 줄이 그어져 있음

선택지 직전 의문:   "저 목록의 줄 그어진 이름들은 죽은 건가, 살아난 건가?"

복선 위치:          타입B — S3에서 세라가 등장할 때
                    같은 '이름 목록 구조'가 세라의 말로 회수됨

친교 씬:            없음
```

## 8. S3 진입 이벤트 — 메리 눈 마주침

```
감정 목표:          배신당한 건지 아닌지 아직 모르는 상태의 불안

압박 단계:          환경 → 감옥 이동 복도. 사슬 소리. 습기와 곰팡이 냄새
                    사회 → 반대 방향에서 메리 등장. 손에 서류 들려 있음
                           (S1-A 선택 시) 메리, 플레이어 보고 즉시 시선 내림
                           (S1-A 미선택 시) 메리, 짧게 고개 끄덕임
                    내적 → 두 경우 모두 메리는 말하지 않음. 그게 더 많은 정보를 줌

NPC 충돌 구조:      간수(이동 통제) ↔ 메리(통로를 지나는 자)
                    간수는 메리를 신경 쓰지 않음
                    메리와 플레이어만 이 순간을 공유
                    간수가 "가시오" 밀면, 플레이어는 메리 눈에서 읽은 것을 확인 못 함

직접 말하지 않는 것: 메리가 증인으로 나올 것이라는 것
                     → 메리 손에 서류가 들려 있음
                     그 서류가 무엇인지
                     → 복도 조명 때문에 내용을 읽을 수 없음

선택지 직전 의문:   없음 (이벤트 씬)
                    플레이어에게 남는 것: "메리는 저 서류를 어디에 가지고 가는 건가"

복선 위치:          타입B — 최종관문에서 메리가 증인석에 서는 장면으로 회수

친교 씬:            없음
```

## 9. S3 감옥 씬 — 세라 등장 / 선택지까지

```
감정 목표:          처음 만났는데 이 사람이 지금 가장 믿을 만하다는 이상한 안도감

압박 단계:          환경 → 감옥 내부. 짚 냄새와 녹슨 쇠 냄새. 어둠
                           다른 수감자들이 플레이어 곁에 오지 않음
                    사회 → 세라가 벽에 등 기대고 앉아 있음. 처음엔 플레이어를 보지 않음
                           (S2-A 선택 시) 심문관 부하 진입 → "공범 이름 세 명 더 대시오"
                           플레이어가 가장 무너질 것 같은 순간, 세라가 처음 말을 건넴
                    내적 → 세라는 벽을 보며 말함. 플레이어를 보지 않음
                           그런데 말이 정확하다

NPC 충돌 구조:      심문관 부하(외부 압박) ↔ 세라(내부 방향 제시)
                    부하 퇴장 후, 세라가 처음으로 플레이어 방향으로 몸을 틂
                    직접 보지는 않음. 방향만 바꿈
                    플레이어는 그 방향 전환에서 정보를 읽어야 함

직접 말하지 않는 것: 세라가 이미 살아남은 방법을 쓴 사람이라는 것
                     → 손목에 쇠고랑 자국 헌 것 + 새 것이 겹쳐 있음
                     세라의 제안이 도덕적으로 무겁다는 것
                     → 말을 다 하고 나서 세라가 한 번도 플레이어를 보지 않음
                     (S2-A 선택 시) 세라의 첫 발화가 다름
                     → "당신 이미 한 번 팔았군요"

선택지 직전 의문:   "'이미 죽은 사람 이름'이 내 상황에서 가능한가?"

복선 위치:          타입D — confession_strategy 획득 시
                    세라가 아무것도 건네지 않음. 말만 함
                    플레이어가 기억해야 하는 것

친교 씬:            세라가 쇠고랑 소리 줄이려 손목 조절하는 버릇
                    플레이어가 그 동작을 주시함 → 세라 눈치채고 멈춤
                    "보지 마." 끝. 명령인지 부탁인지 알 수 없음
```

```
감정 목표:          살아남고 싶다는 의지와 그러려면 무언가를 포기해야 한다는 충돌

압박 단계:          환경 → 법정. 나무 바닥. 방청석 시선이 한꺼번에 플레이어에게
                           코윈이 맞은편에. 지도가 이미 펼쳐져 있음
                    사회 → 코윈이 발언 전 안경 고쳐 씀 → 방청석이 조용해짐
                           메리가 증인석 대기 의자에. 시선 바닥 고정
                           (S1-A 선택 시) 메리 손에 서류 있음
                           (S1-A 미선택 시) 메리 손 비어 있음
                    내적 → 코윈 손가락이 강변을 짚음. 떼지 않음

NPC 충돌 구조:      코윈(재판관 — 토지 목적) ↔ 아비게일(방청석 폭탄 — 발작 가능성)
                    둘이 직접 충돌하지 않음
                    코윈이 압박을 가하는 순간 아비게일이 손을 무릎에 얹음
                    플레이어는 코윈과 아비게일을 동시에 시야에 두어야 함
                    메리는 그 사이 어딘가에 있음. 이름이 불릴 때까지 시선 안 들음

직접 말하지 않는 것: 코윈의 진짜 목적이 토지라는 것
                     → 코윈이 '마녀'라는 단어 말하기 전에 반드시 지도를 한 번 훑음
                     아비게일이 발작을 제어할 수 있다는 것
                     → 법정에서 아비게일은 조용함. 그것이 가장 이상함
                     confession_strategy 사용 시 누가 끌려갈 것인지
                     → 이름을 대는 순간 그 인물이 어디 앉아 있는지 포착

선택지 직전 의문:   "코윈이 손가락을 지도에서 떼지 않는 건 왜지?"

복선 위치 (다중 회수):
                    타입A — S1-B의 "강변 쪽" 두드리는 소리
                           → 코윈이 지도 훑는 동작으로 회수
                    타입A — S3 세라의 "이미 죽은 사람 이름"
                           → confession_strategy 발동 시 회수
                    타입D — 오프닝 아비게일 손가락이 허공 가리킨 것
                           → 법정에서 아비게일이 조용히 손을 무릎에 얹는 것으로 역전 회수

친교 씬 (코윈 — land_deed_info 제출 시):
                    코윈이 서류를 받아들고 멈춤
                    안경을 고쳐 쓰지 않음. 그것이 유일한 신호
                    서류를 내려놓음. 손가락이 지도를 떠남
                    처음으로 지도에서 손가락을 뗌

분기 결과 비트:
                    A (결백 주장): 코윈, 서류 정렬 계속. 멈추지 않음
                    B (confession_strategy): 이름 대는 순간 그 인물 위치 포착
                    C (land_deed_info): 코윈 손가락이 지도를 떠남
```
