*Step 4 대본 집필 전 씬 구조 설계서 — 2026-07-06 전면 재설계*

---

## 클코 집필 지침

이 파일은 대본을 쓰기 전 씬의 뼈대를 정의한다.
각 항목을 읽고, 대본에서 **행동과 대사로 구현**하라.
항목 문장을 대본에 직접 인용하지 않는다.
살을 붙이되, 항목에 명시된 것은 반드시 포함한다.

**항목별 주의:**
- `절차적 상태` = 씬 시작 → 끝의 법적/사회적 상태. 상태가 바뀌면 그 전환 사건이 씬 안에 있어야 한다.
- `인과 연결` = 이 씬이 일어나는 이유(직전 사건). 대본이 이 연결을 플레이어가 체감하게 만들어야 한다.
- `선택지 직전 의문` = 설계 목표. 이 문장을 NARR/THINK에 그대로 쓰지 말 것.
- `복선 위치` = 심는 위치. 복선임을 강조하거나 NARR로 설명하지 말 것. 배경처럼 묻혀야 한다.
- `직접 말하지 않는 것` = 절대 대사/NARR로 설명 금지. 행동·표정·소리로만.
- **복선 반복 금지** = 같은 복선 소재(단어/소품)는 한 씬에서만. THINK가 눈치채는 것까지는 허용,
  의미를 해석해 주는 것은 금지. (❌ "왜 강변을 봤지. 거긴 우리 땅인데" / ✅ "왜 강변을 봤지")
- **`대화 구조` = 플레이어–NPC 왕복 설계** (target_tone_sample.md 패턴 7).
  NPC가 던지고 플레이어가 THINK로만 받는 일방 구조 금지. 플레이어는 말한다(PLAYER 태그),
  NPC는 플레이어의 직전 발화에 반응한다. 말할 수 있는 것은 말하게 하고, THINK에는
  말할 수 없는 것(의심·공포·계산)만 남긴다. NPC 티키타카 중 오르페아 개입은 씬당 최대 1회.
  - **'강변/땅' 단어 등장 허용 씬**: Beat 3(메리 시선만 — 단어 발화 없음) / Beat 5(코윈 목소리 1회) / 최종관문(지도).
    그 외 씬에서 땅 얘기 금지. B분기 내부에서만 예외적으로 답이 플레이어에게 닿는다(아래 Beat 5 참조).

## 에피소드 기승전결 지도

```
기  Beat 0~2   오프닝     접속 → 발작·지목 → 소동 후 거리두기. 평온이 깨지는 순간.
승  Beat 3~7   스테이지1  정식 고발(위협 구체화) → 체포 전 마지막 하루의 대응(선택) → 한밤 체포(대응 무력화 = 절정)
전  Beat 8~11  스테이지2~3 제도의 올가미(심문) → 감옥 → 탈출 열쇠(세라)
결  Beat 12    최종관문   법정. 모든 복선 회수. 선택의 대가.
```

**절차적 상태 흐름 (한 칸씩만 이동, 전환마다 씬 존재):**
```
자유민 → 공개 지목됨(Beat 1) → 정식 고발됨(Beat 3) → 체포됨(Beat 7) → 심문받음(Beat 9) → 재판 대기 구금(Beat 10~11) → 본 재판(Beat 12)
```

**역사 절차 근거 (fact_sheet + 1692 실제 절차):**
공개 소동 → 성인 남성 명의 고발장(complaint) 접수 → 치안판사(호손·코윈) 영장 발부 → 체포 →
예비심문(examination, 호손·코윈 주재) → 구금 → Oyer and Terminer 법정 재판.
고발장 접수부터 체포까지 통상 1~2일 — 스테이지1의 "하루"라는 시간 압박이 여기서 나온다.

## 기 — 오프닝

### Beat 0. 접속 시네마틱
**사용자 확정본 유지** (vn_script.md opening_cinematic — 신경 동기화 88% → 98.7% → 100.0%).
수정하지 않는다. 이 톤이 에피소드 전체의 목표 톤 샘플이다.

### Beat 1. 예배당 — 신분 부여 + 발작 + 직접 지목

```
절차적 상태:        자유민 → 공개 지목됨 (법적 절차는 아직 없음 — 소동일 뿐)
인과 연결:          에피소드 시작점. 접속 직후.

감정 목표:          유머 섞인 혼란 → 한순간에 얼어붙는 공포 (30초 전복)

집필 기준:          opening_01(오르페아 티키타카·신분 부여)은 사용자 확정본 유지.
                    opening_02(발작·지목)는 사용자 확정본 골격 유지 + 오탈자 정리.

압박 단계:          환경 → 연기·땀·짐승 냄새. 낮은 천장. 빼곡한 사람들
                    사회 → 설교 후 아이들이 연단 앞으로. 시선 집중
                    내적 → 눈 마주침 → 발작 → "엘리자베스!" 이름 직접 지목

NPC 충돌 구조:      아비게일(발작 중심) ↔ 군중(공포 감염) 사이에 플레이어
                    아비게일은 플레이어를 이름으로 직접 지목한다 (사용자 확정)
                    "어젯밤 제 방에 왔어요" — 숙주 엘리자베스와 아는 사이라는 함의

발작 묘사 원칙:     발작은 구체적·감각적으로 묘사한다 (비명, 눈 뒤집힘, 핏대, 경련).
                    금지는 단 하나 — '연기'라는 해석. 발작 직전 눈 초점 소실은
                    SHOW: abigail vacant 표정 컷으로만. NARR 설명 금지.

직접 말하지 않는 것: 아비게일이 연기라는 것 → vacant 표정 컷 0.5초로만
                     플레이어(빙의 의식)와 숙주(엘리자베스)의 기억 비대칭
                     → THINK "누구지. 아는 애인가?" 수준까지만. 오르페아 설정 강의 금지

복선 위치:          없음 (Beat 2로 이동)
친교 씬:            없음
```

### Beat 2. 예배당 밖 — 소동의 뒷맛 + 아들 재회

```
절차적 상태:        공개 지목됨 (변화 없음 — 아무도 그 자리에서 잡아가지 않는다.
                    1692년엔 체포에도 서류가 필요하다. 이 '아직 아무 일도 없음'이 다음 비트의 낙차를 만든다)
인과 연결:          Beat 1의 발작·지목 직후. 예배가 파하고 사람들이 나온다.

감정 목표:          아무 일도 안 일어났다는 안도 + 등 뒤에서 세계가 재배열되는 감각

압박 단계:          환경 → 밖. 햇빛이 너무 밝다. 군중이 흩어진다
                    사회 → 아내들이 말없이 남편 곁으로 바짝 붙는다. 한 명씩.
                           인사하던 이웃이 눈을 내리깔고 지나간다.
                           ※ 이것은 단순 거리두기가 아니라 사회적 평판이 추락하는 '순간'이다.
                             어제까지의 이웃이 오늘 다른 생물이 되는 것 — 무게 있게, 그러나 행동으로만.
                    내적 → 돌담 곁의 아들. 작은 돌을 발끝으로 굴리고 있다. 재회.
                           아이는 소동을 모른다. 그 무지가 아프다.

NPC 충돌 구조:      마을 여론(침묵의 거리두기) ↔ 아들(무구함) 사이에 플레이어

직접 말하지 않는 것: 재산 있는 혼자인 여성의 위험성 → 아내들이 남편 곁으로 붙는 행동으로만
                     플레이어의 취약성 → 아들의 손을 잡는 순간의 체온 대비로만

복선 위치:          타입D — 흩어지는 군중 사이, 검은 코트의 남자가 서기와 몇 마디 나누고
                    마을회관 방향으로 걸어간다. 아무도 주목하지 않는다. 얼굴은 보이지 않는다.
                    (회수: Beat 9 심문실에서 같은 검은 코트 → 최종관문에서 완성)

대화 구조:          아들과 2왕복. 아이가 먼저 말을 건다(소동을 모르는 천진한 질문 —
                    "오늘 왜 이렇게 일찍 끝났어?" 류). 플레이어의 대답이 어긋난다 —
                    이 아이의 엄마로 살아온 기억이 없으니까. 아이는 그 어긋남을 눈치 못 챈다.
                    그 무지가 아프다. THINK는 대답 사이의 당혹("얘한테 뭐라고 불려야 하지")만.

친교 씬:            아들과의 짧은 순간. 정보 전달 없음.
                    아이가 주운 돌을 주머니에 넣어주는 것 같은 사소한 동작 1개.
                    (이 온기가 Beat 7 체포의 타격을 만든다 — 반드시 심을 것)
```

## 승 — 스테이지1: 고발장, 체포 전 마지막 하루

### Beat 3. S1 진입 — 메리의 소식 + 선택지

```
절차적 상태:        공개 지목됨 → 정식 고발됨 (전환 사건 = 메리가 전하는 고발장 접수 소식)
인과 연결:          Beat 1의 공개 지목이 하룻밤 새 마을 소문망을 돌아
                    퍼트넘가 명의의 정식 고발장이 됐다. 어제의 소동이 오늘 서류가 된 것.

감정 목표:          "시간이 없다"는 압박. 온기를 가장한 재촉의 찜찜함.

압박 단계:          환경 → 다음 날 아침. 집 마당. 일상적인 풍경 (아이, 닭, 빨래)
                    사회 → 메리가 급히 들어온다. 두리번거림. 속삭임:
                           고발장이 접수됐다. 영장이 곧 나온다. 남은 건 오늘 하루.
                           ※ 메리가 이 정보를 아는 이유 = 그녀는 마을 소문을 누구보다 먼저 듣는 사람
                             (npc_cards: 두리번거림, 빠른 말, 정보 수집 버릇으로 표현)
                    내적 → 메리의 말이 전부 맞다. 너무 맞아서, 너무 빨라서 무섭다.

NPC 충돌 구조:      메리(접근·재촉) ↔ 보이지 않는 시계(영장) 사이에 플레이어
                    이웃 남성이 지나가며 메리를 흘겨봄 → 메리 0.5초 굳음 → 다시 웃음

플레이어 기억 비대칭 처리:
                    메리는 구면처럼 군다. 플레이어 THINK는 "낯이 익어야 할 것 같은데
                    아무것도 안 떠오른다" 수준 1회만. 설명 금지. (오프닝에서 확립된 설정의 재확인)

대화 구조:          **이 씬의 핵심 — 최소 3왕복. 메리의 일방 브리핑 금지.**
                    기억 비대칭이 플레이어 질문의 명분이다: 몸은 아는 사이지만 플레이어는
                    아무것도 모르니까 물을 수밖에 없다.
                    설계 예시 (그대로 쓰지 말고 리듬만):
                    메리, 소식 투하 → 플레이어 "고발이요? 누가요?" → 메리, 이름 대며 재촉 →
                    플레이어 "잠깐만요, 그게 무슨 죄가 되는데요?" (현대인의 어긋난 질문 —
                    메리가 0.5초 멈칫: "...그걸 왜 물어?") → 메리, 탄원서 제안 →
                    플레이어 "그게 효과가 있어요?" → 메리 "효과 있을 거야" (눈을 안 맞춤).
                    플레이어의 질문이 메리의 수상한 신호(멈칫, 시선 회피)를 끌어내는 구조 —
                    질문은 정보 획득 수단이자 복선 노출 장치다.
                    오르페아 개입은 1회 이하 (현대인 질문이 어긋났을 때 꼬집는 용도).

선택지 리프레임:    세 선택지 모두 '체포 전 마지막 하루에 할 수 있는 일'이다.
                    A: 탄원서 — 메리의 제안. "마을이 네 편이라는 서류를 만들어. 재판 전에."
                       현대 논리로 가장 합리적 (서명 = 증거 = 안전) → 함정
                    B: 예배 — 3주 빠진 예배. 시대 논리로 맞는 선택 (신앙 증명) → 탈출구
                    C: 고양이 고발 — 의심을 다른 곳으로. 당시엔 정상, 현대엔 황당 → 웃긴 선택지

직접 말하지 않는 것: 메리가 두려워하고 있다는 것 → 말할 때 손이 너무 많이 움직임
                     탄원서가 법적 효력이 없다는 것 → "효과 있을 거야" 말할 때 눈을 맞추지 않음
                     메리가 재산에 관심 있다는 것 → 대화 중 시선이 강변 방향으로 1회 이탈. 말 없음.
                     ※ THINK는 "방금 어딜 본 거지" 질문까지만. '강변'이라는 단어도, '땅'이라는 해석도 금지.

선택지 직전 의문:   "하루 안에 뭘 해야 살 수 있지?"

복선 위치:          타입B — 메리의 강변 방향 시선 (이 씬에서 '강변'은 아무도 발화하지 않음)
친교 씬:            메리가 아들에게 말 없이 건빵 건넴. 아이를 보지 않음.
                    건네는 손과 받는 손이 잠깐 겹침. 메리는 이미 다른 곳을 보고 있음.
```

### Beat 4. S1-A 분기 — 탄원서

```
절차적 상태:        정식 고발됨 (변화 없음 — 대응 행동 중)
인과 연결:          Beat 3에서 A 선택. 고발장이 존재하므로 '반박 서류'라는 대응이 성립한다.

감정 목표:          뭔가를 해냈다는 짧은 안도 → 즉시 오는 서늘함

씬 구성 (2씬):
  4-1. 서명 수집:   메리와 함께 이웃집들을 돈다.
                    문이 안 열리는 집 / 문틈으로만 대답하는 집 / 서명하는 소수 (3~4명).
                    서명하는 사람들의 손이 하나같이 빠르다 — 빨리 끝내고 싶은 손.
                    대화 구조: 문전 왕복 최소 2회 — 플레이어가 직접 부탁하고, 거절이
                    각기 다른 형태로 돌아온다 (말로 하는 핑계 / 문틈의 침묵 / "미안해요" 한마디와
                    닫히는 문). 거절하는 이웃도 플레이어의 말에 반응해야 한다 — 허공 거절 금지.
  4-2. 제출:        서기 창구. 메리가 탄원서를 밀어 넣는다.
                    문 너머 낮고 느린 목소리: "서명인 명단을 올려."
                    메리가 즉시 플레이어 팔에서 손을 뗀다. 한 발 물러선다.
```

```
압박 단계:          환경 → 이동 복도/길. 사슬 소리. 습기와 곰팡이 냄새
                    사회 → 반대 방향에서 메리 등장. 손에 서류 들려 있음
                           (S1-A 선택 시) 메리, 플레이어 보고 즉시 시선 내림
                           (S1-A 미선택 시) 메리, 짧게 고개 끄덕임
                    내적 → 두 경우 모두 메리는 말하지 않음. 그게 더 많은 정보를 줌

NPC 충돌 구조:      간수(이동 통제) ↔ 메리(통로를 지나는 자)
                    간수가 미는 순간, 플레이어는 메리 눈에서 읽은 것을 확인 못 함

직접 말하지 않는 것: 메리가 증인으로 나올 것이라는 것 → 손의 서류로만
                     그 서류가 무엇인지 → 조명 때문에 읽을 수 없음

복선 위치:          타입B — 메리의 서류 (회수: 최종관문 증인석)
친교 씬:            없음
```

### Beat 11. S3 감옥 — 세라

```
절차적 상태:        재판 대기 구금 (변화 없음)
인과 연결:          이송 완료. 재판 전 마지막 구간 — 여기서 얻는 것이 법정의 선택지를 결정한다.

감정 목표:          처음 만났는데 이 사람이 지금 가장 믿을 만하다는 이상한 안도감

압박 단계:          환경 → 감옥. 짚 냄새와 녹슨 쇠 냄새. 어둠. 다른 수감자들이 곁에 오지 않음
                    사회 → 세라가 벽에 등 기대고 앉아 있음. 처음엔 플레이어를 보지 않음
                           (S2-A 선택 시) 심문관 부하 진입 → "공범 이름 세 명 더 대시오"
                           플레이어가 가장 무너질 것 같은 순간, 세라가 처음 말을 건넴
                    내적 → 세라는 벽을 보며 말함. 그런데 말이 정확하다

NPC 충돌 구조:      심문관 부하(외부 압박) ↔ 세라(내부 방향 제시)
                    부하 퇴장 후, 세라가 처음으로 플레이어 방향으로 몸을 틂. 직접 보지는 않음

대화 구조:          세라식 티키타카 — 플레이어의 말은 길고, 세라의 답은 짧게 끊는다.
                    플레이어 질문 2~3개에 세라는 단답 ("그렇게 하면 살아. 안 하면 죽어").
                    플레이어의 현대적 반박("그건 거짓말이잖아요")에 세라가 유일하게
                    문장 두 개로 답하는 순간을 만든다 — 그 순간이 이 씬의 무게중심.
                    감사 인사는 "됐어"로 끊김 (npc_cards 버릇). 왕복의 비대칭 자체가 세라의 캐릭터다.

직접 말하지 않는 것: 세라가 이미 살아남은 방법을 쓴 사람이라는 것
                     → 손목에 쇠고랑 자국 헌 것 + 새 것이 겹쳐 있음
                     세라의 제안이 도덕적으로 무겁다는 것
                     → 말을 다 하고 나서 세라가 한 번도 플레이어를 보지 않음
                     (S2-A 선택 시) 세라의 첫 발화: "당신 이미 한 번 팔았군요"

선택지 직전 의문:   "'이미 죽은 사람 이름'이 내 상황에서 가능한가?"
복선 위치:          타입D — confession_strategy: 세라는 아무것도 건네지 않음. 말만 함.
친교 씬:            세라가 쇠고랑 소리 줄이려 손목 조절하는 버릇.
                    플레이어가 주시 → 세라 눈치채고 멈춤. "보지 마." 끝.
```

### Beat 12. 법정

```
절차적 상태:        재판 대기 → 본 재판 (전환 사건 = 개정. Oyer and Terminer)

                    전부 이 방으로 모인다.

감정 목표:          살아남고 싶다는 의지와 그러려면 무언가를 포기해야 한다는 충돌

압박 단계:          환경 → 법정. 나무 바닥. 방청석 시선이 한꺼번에 플레이어에게.
                           코윈이 맞은편에. 지도가 이미 펼쳐져 있음
                    사회 → 코윈이 발언 전 안경 고쳐 씀 → 방청석이 조용해짐 (Beat 9 버릇 회수)
                           메리가 증인석 대기 의자에. 시선 바닥 고정
                           (S1-A 선택 시) 메리 손에 서류 / (미선택 시) 손 비어 있음
                    내적 → 코윈 손가락이 지도 위 강변을 짚음. 떼지 않음 (Beat 5 두드림 회수)

NPC 충돌 구조:      코윈(재판관 — 토지 목적) ↔ 아비게일(방청석 폭탄 — 발작 가능성)
                    둘은 직접 충돌하지 않음. 코윈이 압박하는 순간 아비게일이 손을 무릎에 얹음
                    메리는 그 사이 어딘가에. 이름이 불릴 때까지 시선 안 듦

직접 말하지 않는 것: 코윈의 진짜 목적이 토지라는 것
                     → '마녀'라는 단어를 말하기 전에 반드시 지도를 한 번 훑는 동작으로만
                     아비게일이 발작을 제어할 수 있다는 것
                     → 법정에서 아비게일은 조용함. 그것이 가장 이상함
                     confession_strategy 사용 시 누가 끌려갈 것인지
                     → 이름을 대는 순간 그 인물이 어디 앉아 있는지 포착

선택지 직전 의문:   "코윈이 손가락을 지도에서 떼지 않는 건 왜지?"

복선 회수 (다중):   타입A — Beat 5 "강변 쪽" 두드림 → 지도 훑는 동작
                    타입A — Beat 11 "이미 죽은 사람 이름" → confession_strategy 발동

                    서류를 내려놓음. 손가락이 처음으로 지도를 떠남.

분기 결과 비트:     A (결백 주장): 코윈, 서류 정렬 계속. 멈추지 않음
```

## verification_notes

- **탄원서 타이밍**: 실제 세일럼 탄원서(예: Rebecca Nurse, 39인 서명)는 체포 후 제출이 다수.

  역사적 개연성 범위 내 단순화.

  후견/신탁 등 법적 디테일은 대본에서 언급하지 않고 넘어간다.

- **고발장 접수 주체**: 퍼트넘가 명의 — Thomas Putnam이 다수 고발장을 접수한 실제 기록 기반.
